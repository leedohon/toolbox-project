사용 방법

  1. 만들 UUID 개수를 1~100 사이로 입력합니다.
  2. 기본, 대문자 또는 하이픈 제거 출력 형식을 선택합니다.
  3. 새 UUID를 만든 뒤 전체 결과를 복사하거나 텍스트 파일로 저장합니다.

UUID 일괄 생성기 상세 설명

UUID 일괄 생성기는 테스트 데이터, 문서 식별자와 임시 레코드 키가 여러 개 필요할 때 사용합니다. 브라우저의 암호학적 난수 기능으로 표준 UUID v4 값을 만들고 한 줄에 하나씩 정리합니다.

생성 결과는 서버로 전송하거나 저장하지 않습니다. UUID는 충돌 가능성이 매우 낮은 식별자이지만 비밀번호나 인증 토큰을 대신하지 않으며, 보안 비밀값에는 목적에 맞는 별도 생성 방식을 사용해야 합니다.

자주 묻는 질문

UUID v4는 무엇인가요?

난수를 기반으로 만드는 128비트 식별자 형식이며 여러 시스템에서 레코드를 구분할 때 사용합니다.

같은 UUID가 나올 수 있나요?

이론적으로 가능하지만 확률은 매우 낮습니다. 도구는 한 번에 만든 결과 안에서도 중복을 확인합니다.

UUID를 비밀번호로 사용해도 되나요?

권장하지 않습니다. UUID는 식별자이며 비밀번호와 인증 토큰은 전용 보안 생성 방식을 사용해야 합니다.

생성 결과가 서버에 저장되나요?

아니요. 난수 생성과 파일 만들기는 현재 브라우저 안에서만 진행됩니다.
