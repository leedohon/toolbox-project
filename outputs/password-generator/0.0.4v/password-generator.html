안전 비밀번호 생성기

길이와 문자 구성을 선택해 브라우저 안에서 안전한 무작위 비밀번호를 만듭니다.

  • 8~64자 길이 조절
  • 소문자·대문자·숫자·특수문자 선택
  • 헷갈리는 문자 제외
  • 암호학적 난수와 복사 대체 UI

안전한 비밀번호 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 원하는 비밀번호 길이를 선택합니다.
  2. 포함할 문자 종류와 헷갈리는 문자 제외 여부를 정합니다.
  3. 생성 결과와 예상 조합 강도를 확인한 뒤 복사합니다.

안전 비밀번호 생성기 상세 설명

안전 비밀번호 생성기는 계정마다 서로 다른 긴 비밀번호가 필요할 때 빠르게 사용할 수 있습니다. 브라우저의 암호학적 난수 기능으로 문자를 고르고 각 선택 문자군이 결과에 포함되도록 구성합니다.

생성된 비밀번호는 서버로 전송하거나 저장하지 않습니다. 중요한 계정에는 생성 결과를 신뢰할 수 있는 비밀번호 관리 앱에 보관하고, 같은 비밀번호를 여러 사이트에서 반복 사용하지 않는 것이 좋습니다.

자주 묻는 질문

비밀번호가 서버에 저장되나요?

아니요. 생성은 현재 브라우저 안에서만 진행되며 결과를 전송하거나 저장하지 않습니다.

몇 자가 적당한가요?

일반 계정은 16자 이상을 권장하며 서비스가 허용한다면 20자 이상이 더 안전합니다.

특수문자를 꼭 포함해야 하나요?

서비스 규칙에 따라 다릅니다. 허용된다면 여러 문자군을 함께 쓰는 편이 조합 수를 늘리는 데 도움이 됩니다.

생성 결과를 그대로 사용해도 되나요?

사이트의 비밀번호 규칙을 확인하고 비밀번호 관리 앱에 안전하게 보관해 사용하세요.
