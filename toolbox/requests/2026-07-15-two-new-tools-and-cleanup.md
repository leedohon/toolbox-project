# 신규 도구 2종과 기존 기능 정리

## 요청

- 실용적인 새 도구 두 개를 자율적으로 선정해 게시한다.
- 기존 도구를 검토하고 의미 없는 기능을 삭제한다.
- 다중팀 편가르기에서 팀 이름 입력 기능을 제거한다.

## 신규 도구 선정

1. URL 인코더·디코더
   - 주소 전체 또는 쿼리 값에 맞는 인코딩과 디코딩을 브라우저에서 처리한다.
   - 입력과 결과는 세로로 배치하고 바꾸기·복사·초기화를 제공한다.
2. 날짜 계산기
   - 두 날짜 사이의 일수 계산과 기준일에서 날짜 더하기·빼기를 제공한다.
   - 종료일 포함 여부와 음수 일수 입력을 지원한다.

## 기존 기능 검토

- 편가르기의 팀 이름 입력은 결과 구분에 필수적이지 않고 입력 단계를 늘리므로 삭제한다.
- 결과 제목은 언어에 따라 `팀 1, 팀 2` 또는 `Team 1, Team 2`로 자동 표시한다.
- 이전 결과 코드의 팀 이름 값은 무시하되 팀 구성은 계속 복원해 호환성을 유지한다.
- QR·바코드 저장·초기화, JSON 표·내보내기, Base64 바꾸기·파일 처리, 놀이 도구 공유·PNG, 모자이크 샘플·되돌리기는 각각 핵심 작업 또는 실패 복구에 직접 필요하므로 유지한다.
- 삭제 판단은 단순히 사용 빈도가 낮아 보인다는 이유가 아니라 핵심 작업과 중복되고 사용자 입력만 늘리는 경우로 제한한다.

## 버전

- URL 인코더·디코더: `0.0.1v`, index 9
- 날짜 계산기: `0.0.1v`, index 10
- 다중팀 편가르기: 팀 이름 제거 기능 변경으로 `0.4.0v`

## 검증

- URL 한글·공백·예약문자 왕복 변환과 잘못된 퍼센트 문자열 오류를 확인한다.
- 날짜 차이, 종료일 포함, 날짜 더하기와 빼기를 확인한다.
- 편가르기 화면·복사·결과 코드·PNG에서 사용자 팀 이름 입력이 제거됐는지 확인한다.
- 세 도구의 KOR / ENG, 375px 모바일과 1280px 데스크톱 화면을 확인한다.
