JSON 도구

JSON을 실시간으로 검사하고 보기 좋게 정리하거나 한 줄로 압축할 수 있습니다.

  • JSON 유효성과 오류 위치 확인
  • 들여쓰기 정리 또는 한 줄 압축
  • 결과 복사와 JSON 파일 저장

JSON 정리하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. JSON을 입력창에 붙여 넣습니다.
  2. 정리 또는 압축 방식을 선택합니다.
  3. 결과를 복사하거나 파일로 저장합니다.

자주 묻는 질문

입력한 JSON이 서버에 저장되나요?

아니요. 모든 처리는 현재 브라우저 안에서 진행됩니다.

기능 업데이트 때 게시글을 다시 수정해야 하나요?

아니요. 고정된 iframe 주소가 최신 기능을 제공합니다.
