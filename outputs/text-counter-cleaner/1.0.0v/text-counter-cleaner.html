자주 묻는 질문

어떤 텍스트 기능을 지원하나요?

글자수·바이트 확인과 공백 정리, 파일명 정리, URL 슬러그, 줄 정렬·중복 제거, Lorem Ipsum 생성을 지원합니다.

정리 기능을 실행하면 원본 글이 바뀌나요?

아니요. 원본 입력은 유지하고 공백·빈 줄·줄바꿈을 정리한 결과를 별도 영역에 표시합니다.

파일명과 URL 슬러그는 어떻게 다르나요?

파일명 모드는 운영체제에서 사용할 수 없는 문자를 정리하고, 슬러그 모드는 제목을 소문자 URL 경로에 알맞은 형태로 바꿉니다.

줄 목록과 Lorem Ipsum은 어떤 상황에 쓰나요?

줄 목록은 후보·태그 같은 값을 정렬하고 중복을 없앨 때, Lorem Ipsum은 화면이나 문서 레이아웃을 시험할 임시 문장이 필요할 때 사용합니다.

빈 입력이나 복사 오류가 나오면 어떻게 하나요?

표시된 빈 값이나 잘못된 행을 채우고 다시 실행하세요. 자동 복사가 차단되면 대체 입력칸의 결과를 직접 선택해 복사할 수 있습니다.

입력한 글이 서버에 저장되나요?

아니요. 모든 집계, 정리, 변환과 문장 생성은 현재 브라우저 안에서만 진행됩니다.
