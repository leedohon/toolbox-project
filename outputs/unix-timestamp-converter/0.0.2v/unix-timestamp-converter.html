Unix 초·밀리초와 읽기 쉬운 지역 날짜·UTC 시간을 서로 변환합니다.

  • 초·밀리초 자동 판별
  • 지역 날짜와 UTC 동시 표시
  • 날짜에서 Unix 초·밀리초 생성
  • 현재 시간 입력과 결과 복사

Unix 시간과 날짜 변환하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. Unix 숫자를 입력하거나 현재 시간 넣기를 누릅니다.
  2. 지역 날짜와 UTC 결과를 확인합니다.
  3. 반대로 날짜와 시간을 선택해 Unix 초·밀리초를 만들고 복사합니다.

Unix 시간 변환기 상세 설명

Unix 시간 변환기는 로그, API, 데이터베이스에 기록된 큰 정수 시간을 사람이 읽는 날짜로 바꿉니다. 10자리 안팎은 초, 13자리 안팎은 밀리초로 자동 판별하며 필요하면 단위를 직접 지정할 수 있습니다.

날짜를 입력하면 해당 기기의 지역 시간대를 기준으로 Unix 초와 밀리초를 함께 계산합니다. 같은 순간의 UTC ISO 값도 표시해 지역 시간대 차이로 인한 혼동을 줄입니다.

자주 묻는 질문

Unix 시간이 무엇인가요?

1970년 1월 1일 UTC부터 흐른 초 또는 밀리초를 나타내는 숫자입니다.

초와 밀리초는 어떻게 구분하나요?

자동 판별은 숫자 크기를 기준으로 정하며 단위 선택에서 직접 고정할 수도 있습니다.

지역 시간이 다른 이유는 무엇인가요?

같은 순간도 기기의 시간대 설정에 따라 지역 날짜와 시각이 다르게 표시됩니다.

음수 Unix 시간도 변환되나요?

네. 지원 범위 안에서는 1970년 이전 날짜를 나타내는 음수도 변환합니다.
