사용 방법

  1. CSV 또는 TSV 텍스트를 입력하거나 파일을 선택합니다.
  2. 자동 판별된 구분자와 첫 행 제목 사용 여부를 확인합니다.
  3. 표 미리보기를 확인한 뒤 JSON, CSV 또는 TSV로 복사·저장합니다.

CSV 표 보기·변환기 상세 설명

CSV 표 보기·변환기는 스프레드시트나 업무 시스템에서 받은 쉼표·탭 구분 데이터를 브라우저에서 바로 확인할 때 사용합니다. 따옴표 안의 쉼표와 줄바꿈을 구분해 읽고 첫 행을 열 이름으로 사용할지 선택할 수 있습니다.

파일과 텍스트는 서버에 업로드하지 않고 현재 브라우저에서만 처리합니다. 표는 성능을 위해 처음 100행을 미리 보여 주지만 변환 파일에는 입력된 전체 행을 포함하며, 열 개수가 맞지 않는 행은 가까운 오류 안내로 알려 줍니다.

자주 묻는 질문

어떤 구분자를 지원하나요?

쉼표, 탭, 세미콜론과 세로 막대를 지원하며 자동 판별하거나 직접 선택할 수 있습니다.

따옴표 안의 쉼표도 처리되나요?

네. 큰따옴표로 감싼 셀 안의 쉼표와 줄바꿈은 하나의 값으로 처리합니다.

CSV 파일이 서버로 올라가나요?

아니요. 파일 읽기, 표 생성과 변환은 현재 브라우저 안에서만 진행됩니다.

행마다 열 개수가 다르면 어떻게 되나요?

첫 행의 열 개수와 다른 행 번호를 표시하므로 원본 데이터를 바로 고칠 수 있습니다.
