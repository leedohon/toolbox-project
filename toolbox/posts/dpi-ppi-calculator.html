사용 방법

  1. 이미지의 가로·세로 픽셀을 입력합니다.
  2. 출력할 실제 크기와 inch 또는 cm 단위를 선택합니다.
  3. 계산된 가로·세로 DPI와 평균값을 확인합니다.

DPI·PPI 해상도 계산기 상세 설명

DPI·PPI 계산기는 이미지의 픽셀 수를 인쇄 또는 화면의 실제 길이로 나눠 밀도를 구합니다. 같은 이미지라도 출력 크기가 작을수록 인치당 픽셀 수가 높아집니다.

계산값은 이미지의 픽셀과 목표 크기에 따른 이론적인 밀도입니다. 프린터 품질, 이미지 보간, 화면 배율 같은 실제 출력 조건은 별도로 확인해야 합니다.

자주 묻는 질문

DPI와 PPI는 같은 뜻인가요?

엄밀히는 인쇄 점과 화면 픽셀을 가리키지만 이미지 크기 계산에서는 관행적으로 비슷하게 사용됩니다.

cm를 입력해도 되나요?

네. cm를 선택하면 내부에서 inch로 바꿔 계산합니다.

300 DPI면 항상 인쇄 품질이 좋은가요?

일반적인 기준이지만 원본 선명도와 프린터·종이 조건도 결과에 영향을 줍니다.

가로와 세로 DPI가 다르면 문제인가요?

입력한 픽셀 비율과 실제 크기 비율이 다르다는 뜻이며 이미지가 늘어날 가능성을 확인해야 합니다.
