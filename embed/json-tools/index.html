JSON 도구

JSON을 실시간으로 검사하고, 보기 좋게 정리하거나 한 줄로 압축할 수 있습니다. 모든 처리는 브라우저 안에서만 진행됩니다.

이 도구로 할 수 있는 일
  • 잘못된 JSON을 줄·열 위치와 함께 즉시 확인
  • 들여쓰기가 적용된 읽기 쉬운 JSON으로 정리
  • 공백을 제거한 한 줄 JSON으로 압축
  • 결과 복사, JSON·Excel 파일 저장, 입력과 결과 비우기

JSON 정리하기

결과 방식

JSON을 입력하면 실시간으로 검사합니다.

유효한 JSON을 입력하면 결과가 여기에 표시됩니다.

Excel 표 미리보기

유효한 JSON을 입력하면 표를 만듭니다.

표시할 데이터가 없습니다.

사용 방법

  1. 상단 입력창에 JSON을 붙여 넣거나 직접 입력합니다.
  2. 유효성 검사 결과를 즉시 확인하고, 보기 좋게 정리 또는 한 줄 압축을 선택합니다.
  3. 아래쪽 결과를 복사하거나 JSON·Excel 파일로 저장합니다.

자주 묻는 질문

입력한 JSON이 서버에 저장되나요?

아니요. JSON 확인, 정리, 압축, 파일 저장은 모두 현재 브라우저 안에서만 처리됩니다.

오류 위치는 어떻게 확인하나요?

JSON이 올바르지 않으면 오류 메시지에 줄과 열 번호를 표시합니다. 해당 위치 부근의 쉼표, 따옴표, 괄호를 확인해 보세요.

정리와 압축의 차이는 무엇인가요?

정리는 줄바꿈과 들여쓰기를 추가해 읽기 쉽게 만들고, 압축은 불필요한 공백을 제거해 한 줄로 만듭니다.
