  • 안전한 HTML 변환·저장
  • 행·열 기반 표 문법 생성
  • 결과 복사와 KOR / ENG

마크다운 문서 또는 표 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 문서 미리보기·HTML 변환 또는 표 생성 모드를 선택합니다.
  2. 마크다운 문서나 표의 머리글·행 데이터를 입력합니다.
  3. 미리보기와 결과 문법을 확인한 뒤 복사하거나 저장합니다.

마크다운 작성 도구 상세 설명

마크다운 작성 도구는 문서를 실시간으로 미리 보고 안전한 HTML로 변환하는 모드와 행·열 데이터를 마크다운 표 문법으로 만드는 모드를 한곳에 통합했습니다. README, 블로그 초안과 표가 포함된 문서를 작성할 때 필요한 기능을 선택해 결과를 바로 복사하거나 저장할 수 있습니다.

미리보기 모드는 제목, 강조, 링크, 인용, 목록과 코드 블록을 변환하고 직접 입력한 실행형 HTML은 코드로 다룹니다. 표 생성 모드는 머리글과 행 데이터를 받아 열 수를 맞추고 구분선까지 포함한 마크다운 표를 만들므로 반복해서 파이프 문자를 입력할 필요가 없습니다.

입력한 원문과 표 데이터는 현재 브라우저 안에서만 처리되며 서버로 전송되지 않습니다. 열 개수가 맞지 않거나 빈 항목이 있으면 표시된 행을 고쳐 다시 생성할 수 있고, 복사가 차단되면 대체 결과를 직접 선택하거나 HTML 파일로 저장할 수 있습니다.

자주 묻는 질문

어떤 마크다운 기능을 지원하나요?

문서 미리보기·안전한 HTML 변환과 행·열 데이터를 이용한 마크다운 표 생성을 지원합니다.

미리보기에서 어떤 문법을 사용할 수 있나요?

제목, 굵게, 기울임, 링크, 인라인 코드, 코드 블록, 인용문과 순서·비순서 목록을 지원합니다.

마크다운 표의 열 개수가 맞지 않으면 어떻게 하나요?

머리글과 각 행의 항목 수를 같게 맞추고 빈 셀을 확인한 뒤 다시 생성하세요. 오류가 난 행은 가까운 위치에서 안내합니다.

HTML 태그나 스크립트가 실행되나요?

직접 입력한 HTML은 실행하지 않고 코드로 처리하며 변환기가 생성한 제한된 안전 태그만 미리보기에 사용합니다.

입력한 문서와 표가 서버에 저장되나요?

아니요. 미리보기, HTML 변환과 표 생성은 현재 브라우저 안에서만 진행됩니다.

복사가 되지 않을 때 결과를 어떻게 가져오나요?

자동 복사가 차단되면 화면의 대체 결과를 직접 선택하세요. HTML 변환 결과는 파일로 저장할 수도 있습니다.
