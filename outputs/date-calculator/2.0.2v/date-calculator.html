다용도 계산기 상세 설명

다용도 계산기는 사칙연산과 길이·넓이·부피·무게·속도·시간·데이터 단위, 날짜 계산에 Unix 시간, 화면 비율, 백분율, 평균·중앙값 계산을 더한 통합 도구입니다. 필요한 계산 종류를 먼저 선택하면 각 모드에 맞는 입력만 표시되어 복잡한 설정 없이 바로 결과를 확인할 수 있습니다.

단위와 날짜 모드는 서로 다른 단위의 값과 날짜 차이·더하기·빼기를 처리합니다. Unix 시간 모드는 초·밀리초와 지역 날짜·UTC를 변환하고, 화면 비율은 한쪽 길이에 맞는 반대쪽 길이를 계산합니다. 백분율은 비율·구성비·증감률을, 통계 모드는 숫자 목록의 평균·중앙값·합계·범위를 제공합니다.

계산은 현재 브라우저 안에서만 실행되며 입력값을 서버에 저장하지 않습니다. 숫자가 비어 있거나 날짜·비율·목록 형식이 맞지 않으면 해당 모드에서 수정할 항목을 안내하므로 값을 고친 뒤 다시 계산하거나 초기화할 수 있습니다.

자주 묻는 질문

어떤 계산 모드를 지원하나요?

사칙연산·단위·날짜, Unix 시간, 화면 비율, 백분율, 평균·중앙값 모드를 지원합니다.

Unix 초와 밀리초를 자동으로 구분하나요?

선택한 단위에 따라 초 또는 밀리초로 변환하며 지역 날짜와 UTC 결과를 함께 확인할 수 있습니다.

화면 비율과 백분율은 어떤 값을 계산하나요?

화면 비율은 가로·세로 비율과 한쪽 길이에 맞는 반대쪽 길이를 계산하고, 백분율은 비율값·전체 구성비·증가율과 감소율을 계산합니다.

평균·중앙값 목록에 빈 줄이나 잘못된 값이 있으면 어떻게 하나요?

숫자가 아닌 항목을 표시하면 해당 값을 고치거나 삭제한 뒤 다시 계산하세요. 쉼표나 줄바꿈으로 숫자를 구분할 수 있습니다.

서로 다른 단위와 날짜도 계산할 수 있나요?

네. 값을 같은 결과 단위로 환산해 계산하며, 날짜 차이와 기준일에서 날짜 더하기·빼기도 지원합니다.

입력한 숫자와 날짜가 서버에 저장되나요?

아니요. 모든 계산과 변환은 현재 브라우저 안에서만 처리되며 새로고침하거나 초기화하면 입력 상태를 다시 시작할 수 있습니다.
