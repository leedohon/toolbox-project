도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 일반 계산, 단위 계산, 날짜 계산 중 하나를 선택합니다.
  2. 값과 연산자, 필요한 단위 또는 날짜를 입력합니다.
  3. 즉시 표시되는 결과를 확인하고 필요하면 복사합니다.

다용도 계산기 상세 설명

다용도 계산기는 일상적인 숫자 사칙연산부터 서로 다른 단위의 계산까지 한 화면에서 처리합니다. 길이, 넓이, 부피, 무게, 속도, 시간, 데이터 분야에서 자주 쓰는 단위를 선택하면 두 값을 같은 결과 단위로 환산한 뒤 계산합니다.

예를 들어 1m와 10inch를 더하거나 빼고, 결과를 cm 또는 ft로 바로 확인할 수 있습니다. 숫자는 소수점 둘째 자리까지 입력할 수 있고 결과도 둘째 자리에서 정리해 표시합니다. 기존 날짜 차이와 날짜 더하기·빼기 기능도 그대로 제공합니다.

날짜 계산기는 두 날짜 사이의 실제 일수 차이를 확인하거나 기준일에서 원하는 날짜를 빠르게 찾을 때 사용합니다. 마감일, 여행 일정, 보증 기간, 프로젝트 기간을 계산할 때 달마다 다른 일수를 직접 세지 않아도 됩니다.

종료일 포함 여부를 선택할 수 있으며 날짜 더하기·빼기에는 양수와 음수를 모두 사용할 수 있습니다. 계산은 시간대의 영향을 줄이도록 날짜 단위로 처리합니다.

자주 묻는 질문

m와 inch처럼 다른 단위끼리 계산할 수 있나요?

네. 두 값을 선택한 결과 단위로 각각 환산한 뒤 덧셈, 뺄셈, 곱셈, 나눗셈을 적용합니다.

곱셈과 나눗셈의 단위는 어떻게 표시되나요?

곱셈과 나눗셈은 물리 단위의 차원이 달라질 수 있어 선택한 결과 단위 기준 계산값으로 표시합니다. 면적이나 부피 자체를 변환하려면 해당 단위 분야를 선택하세요.

소수는 몇 자리까지 입력할 수 있나요?

입력은 소수점 둘째 자리까지 지원하며 계산 결과도 소수점 둘째 자리에서 반올림합니다.

어떤 단위 분야를 지원하나요?

길이, 넓이, 부피, 무게, 속도, 시간, 데이터 단위를 지원합니다. 각 분야에서 국내외에서 자주 쓰는 단위를 선택할 수 있습니다.

시작일과 종료일이 같으면 며칠인가요?

기본 결과는 0일입니다. 종료일 포함을 선택하면 시작일 하루를 포함해 1일로 계산합니다.

날짜를 빼려면 어떻게 입력하나요?

날짜 더하기·빼기 모드에서 -30처럼 음수를 입력하면 기준일보다 앞선 날짜를 계산합니다.

윤년의 2월 29일도 반영되나요?

네. 브라우저의 표준 날짜 계산을 사용해 윤년과 월별 일수 차이를 자동으로 반영합니다.

휴대폰에서도 같은 결과를 얻을 수 있나요?

네. 모바일과 PC에서 같은 날짜 계산 규칙을 사용하며 별도 설치 없이 브라우저에서 실행됩니다.
