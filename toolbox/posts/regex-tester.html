사용 방법

  1. 정규식 패턴과 필요한 플래그를 입력합니다.
  2. 검사할 텍스트를 입력합니다.
  3. 일치 항목과 강조된 결과를 확인하고 필요하면 복사합니다.

정규식 테스트 도구 상세 설명

정규식 테스트 도구는 패턴을 작성하면서 문장 안에서 어떤 부분이 일치하는지 즉시 확인할 때 사용합니다. g, i, m, s 플래그를 선택할 수 있고 문법이 잘못되면 패턴 입력 가까이에 이유를 표시합니다.

입력과 검사는 현재 브라우저 안에서만 처리됩니다. 매우 복잡한 패턴은 브라우저를 느리게 할 수 있으므로 패턴과 텍스트 길이를 제한하며, 일치 결과도 안전한 개수까지만 표시합니다.

자주 묻는 질문

정규식 플래그는 무엇인가요?

g는 전체 일치, i는 대소문자 무시, m은 여러 줄 기준, s는 줄바꿈까지 점으로 포함하는 옵션입니다.

패턴 양쪽에 슬래시를 입력해야 하나요?

아니요. 슬래시 없이 패턴 내용만 입력하세요.

일치 항목이 너무 많으면 어떻게 되나요?

화면이 느려지지 않도록 최대 200개까지만 표시하고 나머지는 안내합니다.

입력한 텍스트가 서버로 전송되나요?

아니요. 모든 검사는 현재 브라우저에서만 실행됩니다.
