날짜 계산기

두 날짜 사이가 며칠인지 확인하거나 기준일에서 원하는 일수를 더하고 빼세요.

  • 두 날짜 사이 일수 계산
  • 종료일 포함 여부 선택
  • 날짜 더하기·빼기와 요일 표시
  • 결과 복사와 KOR / ENG

날짜 계산하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 두 날짜 사이 또는 날짜 더하기·빼기를 선택합니다.
  2. 날짜와 필요한 일수를 입력합니다.
  3. 계산 결과와 요일을 확인하고 필요하면 복사합니다.

자주 묻는 질문

시작일과 종료일이 같으면 며칠인가요?

기본 결과는 0일이며 종료일 포함을 선택하면 1일로 계산합니다.

날짜를 빼려면 어떻게 하나요?

더하거나 뺄 일수에 -30처럼 음수를 입력하면 됩니다.
