로마 숫자 변환기

1~3999의 아라비아 숫자와 표준 로마 숫자 표기를 서로 변환합니다.

  • 양방향 변환
  • 1~3999 범위
  • 표준 표기 검증
  • 결과 복사

로마 숫자 변환하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 변환 방향을 선택합니다.
  2. 아라비아 숫자 또는 로마 숫자를 입력합니다.
  3. 표준 표기로 변환된 결과를 복사합니다.

로마 숫자 변환기 상세 설명

로마 숫자 변환기는 일반 숫자를 I, V, X, L, C, D, M 조합으로 바꾸거나 반대로 읽습니다. 연도, 장 번호, 시계 문자판 표기를 확인할 때 간단히 사용할 수 있습니다.

로마 숫자는 표준 감산 표기만 허용하며 아라비아 숫자는 전통적인 표현 범위인 1부터 3999까지 지원합니다. 입력과 계산은 브라우저에서만 처리됩니다.

자주 묻는 질문

0도 로마 숫자로 바꿀 수 있나요?

전통적인 표준 로마 숫자에는 0 표기가 없어 1~3999만 지원합니다.

소문자로 입력해도 되나요?

네. 소문자를 대문자로 정리해 검증합니다.

IIII 같은 표기도 허용하나요?

아니요. IV처럼 표준 감산 표기를 사용해야 합니다.

3999보다 큰 수는 왜 안 되나요?

윗줄 기호가 필요한 확장 표기 대신 기본 문자만 사용하는 범위에 집중하기 때문입니다.
