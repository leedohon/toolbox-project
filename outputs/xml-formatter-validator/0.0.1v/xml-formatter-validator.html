XML 검사·정리 도구 상세 설명

XML 검사·정리 도구는 브라우저의 표준 XML 파서를 사용해 태그 닫힘, 속성 따옴표, 문서 구조 오류를 확인합니다. 유효한 문서는 선택한 들여쓰기 폭으로 읽기 좋게 정리하거나 공백을 줄인 한 줄 결과로 바꿉니다.

입력과 변환 결과는 현재 브라우저에서만 처리합니다. 외부 엔티티를 가져오거나 스키마 유효성을 판정하지 않으며, 일반적인 XML 문법과 구조를 빠르게 점검하는 용도에 집중합니다.

자주 묻는 질문

XML 스키마나 DTD도 검증하나요?

아니요. 기본 XML 문법과 문서 구조를 검사하며 외부 스키마 검증은 수행하지 않습니다.

민감한 XML이 서버로 전송되나요?

아니요. 입력과 정리 결과는 현재 브라우저 안에서만 처리합니다.

들여쓰기는 몇 칸을 지원하나요?

읽기 좋은 2칸과 4칸 들여쓰기를 선택할 수 있습니다.

주석과 CDATA는 유지되나요?

네. 표준 파서가 읽은 주석과 CDATA 구간을 결과에 보존합니다.
