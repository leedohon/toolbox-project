# 크리에이트 워크플로 2회 연속 실행

## 요청

- 크리에이트 모드를 두 번 연속 실행해 신규 공개 도구 10개를 완성한다.
- 시스템 종료 예약은 사용하지 않는다.
- 각 실행은 신규 도구 5개, Blogger 공개 발행, Git 푸시, GitHub Pages 배포 확인과 별도 미확인 결과 기록을 포함한다.

## 1차 대상

- 해시 생성기: 텍스트·파일의 SHA-256, SHA-384, SHA-512 계산
- 진법 변환기: 2진수·8진수·10진수·16진수 상호 변환
- 파일명 정리기: 운영체제에서 문제가 되는 문자와 공백 정리
- Lorem Ipsum 생성기: 문단 수와 문장 길이에 맞춘 자리표시 문장 생성
- CSS clamp 계산기: 최소·최대 글자 크기와 뷰포트 범위로 반응형 clamp 식 계산

## 2차 대상

- 유니코드 문자 분석기: 문자를 코드 포인트·UTF-16·HTML 표기와 함께 확인
- JWT 내용 확인기: 서명 검증 없이 헤더와 페이로드를 로컬에서 디코딩
- 데이터 용량 변환기: SI와 IEC 단위 사이의 바이트 용량 변환
- DPI·PPI 계산기: 픽셀과 실제 크기로 해상도 밀도 계산
- CSS 그림자 생성기: 시각적 미리보기와 box-shadow 코드 생성

## 공통 규칙

- 모든 입력과 결과는 현재 브라우저 안에서만 처리한다.
- 공통 KOR / ENG 선택, 복사 대체 처리, 안전한 모바일 포커스, iframe 높이 동기화를 재사용한다.
- 새 게시글은 상세 설명 두 문단 이상, 사용자 FAQ 네 개 이상, 중앙 태그와 검색어 20~30개를 포함한다.
- IFRAME_HEIGHT_STALE, HIDDEN_GRID_OVERRIDE, SHARED_ASSET_CACHE_STALE 회귀 검사를 적용한다.
