사다리게임

참가자와 결과를 입력해 실제 사다리 경로로 무작위 매칭을 만들어 보세요.

  • 2~12명 사다리 생성
  • 실제 연결선과 경로 계산
  • 전체 결과 확인과 다시 섞기

사다리게임 하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 참가자 이름을 한 줄에 한 명씩 입력합니다.
  2. 참가자와 같은 개수의 결과를 입력합니다.
  3. 사다리를 만들고 연결된 결과를 확인합니다.

자주 묻는 질문

같은 참가자가 같은 결과를 계속 받나요?

아니요. 만들기 또는 다시 섞기를 누를 때마다 새로운 사다리 연결선을 만듭니다.

입력한 이름이 저장되나요?

아니요. 모든 처리는 현재 브라우저 안에서만 진행됩니다.

업데이트 때 게시글을 다시 수정해야 하나요?

아니요. 고정된 iframe 주소에서 최신 기능을 불러옵니다.
