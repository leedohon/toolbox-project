# 놀이 도구 4종 추가

## 사용자 지시

- 블로그에 가볍게 즐길 수 있는 사다리게임, 돌림판, 숫자뽑기, 다중팀 편가르기를 추가한다.
- 기존 초간단 툴박스 게시글 테마와 반응형 UI를 그대로 따른다.
- 구현, 게시글 작성, Blogger 공개, Git 커밋·푸시, GitHub Pages 배포 및 공개 화면 디자인 검수까지 처리한다.

## 구현 결정

1. 사다리게임
   - 참가자와 결과를 각각 줄바꿈으로 입력하며 2~12개, 같은 개수로 제한한다.
   - 실제 가로 연결선을 무작위로 만들고 경로를 계산한다.
   - 사다리와 전체 결과를 한 화면에서 확인하고 다시 섞을 수 있다.
2. 돌림판
   - 2~20개 항목을 줄바꿈으로 입력한다.
   - 브라우저 난수로 당첨 항목을 정하고 회전 애니메이션 후 결과를 표시한다.
   - 항목별 색상을 자동 배정하고 다시 돌릴 수 있다.
3. 숫자뽑기
   - 최솟값, 최댓값, 뽑을 개수를 입력한다.
   - 중복 허용과 오름차순 정렬을 선택할 수 있다.
   - 결과를 한 번에 복사할 수 있다.
4. 다중팀 편가르기
   - 2~100명의 이름을 줄바꿈으로 입력한다.
   - 2~10개 팀을 선택해 무작위로 섞고 팀별 인원 차이가 최대 1명이 되도록 배분한다.
   - 팀 이름 접두어를 설정하고 결과를 복사할 수 있다.

## 공통 UI 및 안전 기준

- 공통 놀이 도구 CSS를 한 파일에서 관리해 네 도구의 버튼, 입력, 결과 카드, 상태 문구를 통일한다.
- 모바일은 한 열, 데스크톱은 필요한 영역만 두 열로 제공하며 가로 넘침을 만들지 않는다.
- 난수는 `crypto.getRandomValues`를 사용하고 입력은 서버로 전송하지 않는다.
- 과도한 입력과 범위를 제한하고 오류는 입력 가까이에 명확히 표시한다.
- iframe은 `postMessage`로 높이를 전달해 내부 세로 스크롤을 제거한다.

## 릴리스 및 게시 명령

1. `node scripts/build-tool-catalog.mjs`
2. `node scripts/validate-tool-releases.mjs`
3. `node scripts/blogger-publish-tool-posts.mjs ladder-game wheel-spinner number-picker team-divider`
4. `node scripts/blogger-sync-post-urls.mjs`
5. 관련 파일을 커밋해 `main`에 푸시하고 GitHub Pages 및 공개 Blogger 게시글을 확인한다.

## 디자인 검수

- 네 iframe과 게시글을 390px 모바일 및 데스크톱 화면에서 직접 확인한다.
- 빈 입력, 최소·최대 입력, 긴 이름, 여러 팀, 애니메이션 결과를 확인한다.
- 폭 축소, 가로 넘침, 잘림, 버튼 줄바꿈, 중복 스크롤, 결과 카드 간격이 이상하면 수정 후 다시 검수한다.
- 검수 결과와 검수 중 수정한 내용을 각 `patch-notes.json`에 기록한다.
