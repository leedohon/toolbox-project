파일 확장자 MIME 타입 찾기 상세 설명

파일 확장자 MIME 타입 찾기는 json, html, png, pdf, mp4 같은 대표 확장자를 인터넷 미디어 타입으로 연결합니다. 업로드 허용 목록이나 응답 헤더를 작성할 때 빠르게 참고할 수 있습니다.

확장자는 파일 내용의 실제 형식을 보장하지 않으므로 보안 검증에서는 파일 시그니처와 서버 측 검사를 함께 사용해야 합니다. 조회는 브라우저의 정적 목록에서 수행됩니다.
