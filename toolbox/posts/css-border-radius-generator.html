CSS 모서리 둥글기 생성기 상세 설명

CSS 모서리 둥글기 생성기는 border-radius의 네 위치 값을 시계 방향으로 정리해 한 줄 코드를 만듭니다. 카드나 이미지의 일부 모서리만 다르게 둥글게 만들 때 순서를 헷갈리지 않게 합니다.

모든 값이 같으면 네 값 코드도 같은 결과를 냅니다. 이 도구는 px 단위의 단순한 모서리 설정에 집중하며 결과는 현재 브라우저에서 생성됩니다.

값의 순서는 어떻게 되나요?

왼쪽 위, 오른쪽 위, 오른쪽 아래, 왼쪽 아래의 시계 방향입니다.

모두 같은 값으로 입력해도 되나요?

네. 네 모서리가 같은 크기로 둥글어집니다.

퍼센트 단위도 지원하나요?

현재는 카드와 버튼에 자주 쓰는 px 단위에 집중합니다.

음수 값을 사용할 수 있나요?

아니요. CSS 모서리 반경에 맞춰 0 이상의 값만 허용합니다.
