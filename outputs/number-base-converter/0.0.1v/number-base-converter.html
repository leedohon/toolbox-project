2진수·16진수 진법 변환기

2진수·8진수·10진수·16진수를 정수 정밀도 손실 없이 서로 변환합니다.

  • 2·8·10·16진수 입력
  • BigInt 정밀 변환
  • 네 진법 동시 결과
  • 접두어 표시 선택

진법 변환하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 입력값과 현재 진법을 선택합니다.
  2. 필요하면 0x 같은 접두어 표시를 선택합니다.
  3. 네 가지 진법 결과를 확인하고 복사합니다.

2진수·16진수 진법 변환기 상세 설명

진법 변환기는 같은 정수를 2진수, 8진수, 10진수, 16진수 표기로 나란히 보여 줍니다. 개발 중 비트값이나 색상·주소 값의 정수 표현을 대조할 때 유용합니다.

브라우저의 BigInt를 사용해 일반 숫자형의 안전 정수 범위를 넘는 값도 정밀도 손실 없이 처리합니다. 소수는 지원하지 않으며 선택한 진법에 맞지 않는 문자가 있으면 입력 위치 가까이에서 안내합니다.

자주 묻는 질문

큰 정수도 정확히 변환되나요?

네. BigInt로 처리해 지원 길이 안에서는 자릿수를 반올림하지 않습니다.

음수도 입력할 수 있나요?

네. 맨 앞의 빼기 기호를 유지해 각 진법으로 변환합니다.

0x를 입력해도 되나요?

16진수를 선택했을 때 0x 접두어를 허용하며, 2진수의 0b와 8진수의 0o도 지원합니다.

소수도 변환되나요?

아니요. 이 도구는 정수 진법 변환에 집중합니다.
