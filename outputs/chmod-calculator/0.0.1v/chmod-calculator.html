사용 방법

  1. 소유자·그룹·기타 사용자의 권한을 선택합니다.
  2. 권한 계산을 실행합니다.
  3. 숫자·rwx 표기와 chmod 명령 예시를 확인합니다.

chmod 권한 계산기 상세 설명

chmod 권한 계산기는 유닉스 파일 권한의 읽기 4, 쓰기 2, 실행 1 값을 조합해 세 자리 숫자로 보여 줍니다. 755나 644가 어떤 사용자 권한을 뜻하는지 확인할 때 유용합니다.

표시된 명령은 형식 예시이며 실제 서버 파일을 변경하지 않습니다. 중요한 시스템 파일에 적용하기 전에는 대상 경로와 필요한 최소 권한을 별도로 확인하세요.

자주 묻는 질문

755는 무엇을 뜻하나요?

소유자는 읽기·쓰기·실행, 그룹과 기타 사용자는 읽기·실행 권한을 가집니다.

644는 언제 쓰나요?

일반 파일에서 소유자만 쓰고 나머지는 읽기만 하는 조합으로 자주 사용됩니다.

이 도구가 실제 파일 권한을 바꾸나요?

아니요. 권한 숫자와 명령 예시만 계산합니다.

특수 권한도 지원하나요?

최초 버전은 기본 rwx 세 자리 권한에 집중합니다.
