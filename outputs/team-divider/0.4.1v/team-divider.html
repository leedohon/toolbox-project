다중팀 편가르기

참가자 이름을 무작위로 섞어 2개부터 10개까지 여러 팀으로 균등하게 나눠 보세요.

  • 2~100명, 2~10개 팀 균등 배분
  • 팀 1, 팀 2 형식으로 자동 표시
  • 결과 코드 공유와 같은 팀 구성 복원
  • 결과 복사·KOR / ENG·PNG 저장

편가르기 하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 참가자를 추가하고 팀 개수를 선택합니다.
  2. 팀을 나누면 팀 1, 팀 2 형식으로 결과가 표시됩니다.
  3. 결과 코드를 복사해 보내거나 받은 코드를 붙여넣고 같은 팀을 엽니다.

다중팀 편가르기 상세 설명

다중팀 편가르기는 참가자 이름을 개별 행으로 추가하고 원하는 팀 수를 선택해 인원 차이가 최소가 되도록 무작위 배정합니다. 두 팀뿐 아니라 여러 팀으로 나눌 수 있습니다.

결과 코드를 복사하면 같은 팀 구성을 다시 열 수 있고 결과 화면은 PNG로 저장할 수 있습니다. 팀 이름은 선택 언어에 따라 팀 1 또는 Team 1처럼 자동 표시됩니다.

자주 묻는 질문

인원이 딱 나누어지지 않으면 어떻게 되나요?

팀별 인원 차이가 최대 한 명이 되도록 앞 팀부터 한 명씩 더 배정합니다.

두 팀보다 많이 나눌 수 있나요?

네. 참가자 수 안에서 최대 10개 팀까지 선택할 수 있습니다.

빈 참가자 칸이 있으면 어떻게 되나요?

빈 행을 표시하고 이름을 입력하도록 안내합니다. 필요 없는 행이라면 해당 행의 삭제 버튼을 누르세요.

나눈 결과를 그대로 공유할 수 있나요?

네. 결과 코드를 복사해 보내면 상대방이 같은 팀 구성을 열 수 있고 PNG로도 저장할 수 있습니다.
