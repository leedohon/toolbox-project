마크다운 표 생성기

쉼표로 구분한 머리글과 행 데이터를 마크다운 표 문법으로 바꿉니다.

  • 쉼표 구분 머리글
  • 여러 데이터 행
  • 열 수 자동 맞춤
  • 표 문법 복사

마크다운 표 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 머리글을 쉼표로 구분해 입력합니다.
  2. 한 줄에 한 행씩 셀을 쉼표로 구분해 입력합니다.
  3. 생성된 마크다운 표를 복사합니다.

마크다운 표 생성기 상세 설명

마크다운 표 생성기는 머리글과 행 데이터를 파이프 문법으로 변환해 README, 이슈, 문서에 바로 붙일 수 있게 합니다. 열 구분선과 머리글 구분 행을 자동으로 만듭니다.

행의 셀이 부족하면 빈 칸을 채우고 머리글보다 많은 셀은 제외해 열 수를 맞춥니다. 셀 안의 파이프 문자는 이스케이프하며 데이터는 브라우저에서만 처리합니다.

자주 묻는 질문

셀은 어떻게 구분하나요?

한 행 안에서는 쉼표, 행 사이는 줄바꿈을 사용합니다.

행마다 셀 수가 다르면 어떻게 되나요?

머리글 수에 맞춰 부족한 셀은 비우고 초과 셀은 제외합니다.

파이프 문자를 입력할 수 있나요?

네. 마크다운 표가 깨지지 않도록 역슬래시로 이스케이프합니다.

정렬 문법도 선택할 수 있나요?

최초 버전은 기본 왼쪽 정렬 표 생성에 집중합니다.
