Base64 도구

텍스트와 파일을 브라우저 안에서 Base64로 인코딩하거나 디코딩하세요.

  • 텍스트 인코딩과 디코딩
  • 파일 선택과 드래그 앤 드롭
  • 결과 복사와 파일 저장

Base64 변환하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 인코딩 또는 디코딩 모드를 선택합니다.
  2. 텍스트를 입력하거나 파일을 선택합니다.
  3. 결과를 복사하거나 파일로 저장합니다.

자주 묻는 질문

파일이 서버로 전송되나요?

아니요. 파일과 텍스트는 현재 브라우저 안에서만 처리됩니다.

기능 업데이트 때 게시글을 다시 수정해야 하나요?

아니요. 고정된 iframe 주소가 최신 기능을 제공합니다.
