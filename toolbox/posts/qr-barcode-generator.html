QR · 바코드 생성기

텍스트나 URL을 입력해 QR 코드 또는 Code 128 바코드를 만들고 이미지로 저장하세요.

이 도구로 할 수 있는 일
  • 텍스트와 URL을 QR 코드로 생성
  • Code 128 바코드 생성
  • QR 스타일과 출력 크기 조절
  • PNG 또는 SVG 이미지 저장

QR · 바코드 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 텍스트 또는 URL을 입력합니다.
  2. QR 코드나 바코드와 필요한 옵션을 선택합니다.
  3. 결과를 확인하고 이미지로 저장합니다.

자주 묻는 질문

입력 내용이 서버에 저장되나요?

아니요. 생성 작업은 현재 브라우저 안에서 처리됩니다.

아니요. 도구는 고정된 GitHub Pages 주소에서 최신 기능을 불러옵니다.
