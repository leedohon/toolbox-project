# 두 번째 크리에이트 자동 워크플로

## 확정 입력

- 작업 종류: 기존 공개 도구와 목적이 겹치지 않는 신규 도구 5개 제작
- 대상: 텍스트 비교·차이 확인기, 마크다운 미리보기·HTML 변환기, CSV 표 보기·변환기, UUID 일괄 생성기, CSS 그라데이션 생성기
- 버전 단계: 각 도구 최초 `0.0.1v`
- Blogger 게시: 검증을 통과한 5개 모두 공개
- Git·배포: `main` 커밋·푸시와 GitHub Pages 공개 확인
- 종료 예약: 사용자 지시에 따라 사용하지 않음

## 구현 원칙

- 모든 처리는 브라우저 안에서 수행하며 입력 내용을 서버로 전송하지 않는다.
- 각 도구는 고정 `embed/<tool>/` 진입점과 독립 `tool.js` 모듈로 만든다.
- 공통 반응형 스타일, KOR / ENG 선택, 복사 대체 UI, 모바일 자동 포커스 방지와 실시간 iframe 높이 동기화를 재사용한다.
- 게시글은 하나의 검색 의도, 두 문단 이상의 상세 설명, 사용자 FAQ 4개, 해시태그와 20~30개 검색어를 포함한다.
- 이전 버전과 현재 실행 기록은 삭제하거나 덮어쓰지 않는다.
