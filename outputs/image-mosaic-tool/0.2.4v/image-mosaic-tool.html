이미지 모자이크 도구

사진에서 얼굴, 이름, 번호처럼 숨기고 싶은 부분을 브러시로 칠하거나 사각형으로 선택해 모자이크 처리하세요.

  • 파일 선택과 드래그 앤 드롭
  • 브러시 칠하기·사각형 범위 선택
  • 모자이크 강도 조절과 선택 범위 미리보기
  • 되돌리기·초기화·PNG 저장
  • KOR / ENG 화면 전환

이미지 모자이크 하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. PNG, JPEG 또는 WebP 이미지를 선택하거나 드롭합니다.
  2. 브러시 또는 사각형 선택을 고르고 모자이크 강도를 조절합니다.
  3. 브러시로 숨길 부분을 칠하거나 시작점부터 끝점까지 사각형으로 드래그합니다.
  4. 결과를 확인한 뒤 PNG 파일로 저장합니다.

이미지 모자이크 도구 상세 설명

이미지 모자이크 도구는 사진의 얼굴, 차량 번호, 문서의 개인정보처럼 공개하고 싶지 않은 부분을 브라우저에서 가릴 때 사용합니다. 자유롭게 칠하거나 사각형 범위를 지정해 필요한 위치만 처리할 수 있습니다.

모자이크 강도를 조절하고 적용 단계를 되돌릴 수 있으며 결과는 PNG로 저장합니다. 원본 이미지와 편집 내용은 서버에 업로드하지 않습니다.

자주 묻는 질문

이미지가 서버로 업로드되나요?

아니요. 이미지 불러오기, 모자이크 처리, 저장은 모두 현재 브라우저 안에서 진행됩니다.

자유 칠하기와 사각형 선택은 어떻게 다른가요?

자유 칠하기는 손가락이나 마우스로 지나간 부분을 처리하고, 사각형 선택은 드래그 시작점과 종료점 사이의 네모 영역 전체를 한 번에 처리합니다.

모자이크를 잘못 적용했다면 어떻게 하나요?

한 단계 되돌리기를 누르거나 원본으로 초기화한 뒤 다시 작업할 수 있습니다.

휴대폰 사진도 편집할 수 있나요?

네. 휴대폰에서 사진을 선택하고 손가락으로 드래그해 적용한 뒤 PNG로 저장할 수 있습니다.
