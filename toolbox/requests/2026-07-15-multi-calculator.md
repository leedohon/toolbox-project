# 날짜 계산기 다용도 계산기 개편

## 요청

- 날짜 계산기를 다용도 계산기로 확장한다.
- 일반 사칙연산과 길이, 넓이, 부피, 무게, 속도, 시간, 데이터 단위 계산을 제공한다.
- 서로 다른 단위끼리도 덧셈, 뺄셈, 곱셈, 나눗셈을 실행할 수 있어야 한다.
- 숫자 입력과 결과는 소수점 둘째 자리까지 지원한다.
- 기존 날짜 차이 및 날짜 더하기·빼기 기능을 유지한다.

## 계산 규칙

- 일반 계산은 두 숫자에 사칙연산을 적용한다.
- 단위 계산은 양쪽 값을 사용자가 선택한 결과 단위로 각각 환산한 뒤 사칙연산한다.
- 덧셈·뺄셈은 결과 단위를 그대로 표시한다.
- 곱셈·나눗셈은 물리 단위의 차원이 달라질 수 있으므로 `선택 단위 기준 계산값`으로 표시한다.
- 0으로 나누기, 숫자 누락, 소수점 셋째 자리 이상의 입력은 가까운 상태 영역에서 안내한다.
- 모든 계산은 브라우저 안에서 처리한다.

## UX 및 검증

- 일반, 단위, 날짜의 세 계산 종류는 라디오 선택으로 제공한다.
- 단위 종류가 많은 항목은 셀렉트를 사용한다.
- 입력 변화 시 안전하게 즉시 계산하며 모바일에서 자동 포커스하지 않는다.
- 360px 모바일과 데스크톱에서 가로 넘침, 단위 선택 폭, 버튼 줄바꿈을 확인한다.
- m와 inch 등 서로 다른 단위의 사칙연산, 소수 둘째 자리, 날짜 계산을 검증한다.
