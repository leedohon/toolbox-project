안전한 파일명 정리기

파일명에서 사용할 수 없는 문자와 불필요한 공백을 정리해 복사 가능한 이름으로 바꿉니다.

  • Windows·macOS 호환 문자 정리
  • 확장자 보존
  • 공백·구분자 선택
  • 변경 내용 안내

파일명 정리하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 정리할 파일명을 입력합니다.
  2. 공백을 유지하거나 하이픈·밑줄로 바꿀지 선택합니다.
  3. 미리보기와 변경 내역을 확인한 뒤 복사합니다.

안전한 파일명 정리기 상세 설명

파일명 정리기는 운영체제에서 예약되거나 경로로 해석될 수 있는 문자를 제거하고 연속된 공백과 구분자를 보기 좋게 정리합니다. 문서 공유나 웹 업로드 전에 이름 오류를 줄일 수 있습니다.

파일 내용이나 실제 파일을 변경하지 않고 입력한 이름 문자열만 다룹니다. 확장자는 가능한 한 유지하며 Windows 예약 이름처럼 저장에 실패할 수 있는 경우에는 안전한 접두어를 붙여 안내합니다.

자주 묻는 질문

실제 파일 이름이 바로 바뀌나요?

아니요. 안전한 이름을 만들어 보여 줄 뿐이며 복사 후 직접 이름을 바꾸면 됩니다.

확장자는 유지되나요?

네. 마지막 점 뒤의 일반적인 확장자는 유지하고 파일명 부분을 중심으로 정리합니다.

한글 파일명도 사용할 수 있나요?

네. 한글과 대부분의 유니코드 문자는 유지합니다.

왜 CON 같은 이름이 바뀌나요?

일부 이름은 Windows 예약 장치명이라 저장할 수 없으므로 안전한 접두어를 붙입니다.
