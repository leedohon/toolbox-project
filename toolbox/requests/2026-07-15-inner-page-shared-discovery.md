# 내부 화면 공통 상단 검색

## 요청

- 메인 화면을 제외한 게시글, 라벨 검색, 검색 결과, 보관함 등 모든 내부 화면의 상단 구성을 통일한다.
- 게시글에 적용된 Openworld Blog 소개와 JSON 도구 검색을 나머지 내부 화면에도 동일하게 제공한다.

## 구현

- `#ow-site-app`이 있는 메인만 홈 화면으로 판별하고 그 외 화면은 공통 내부 화면으로 처리한다.
- 내부 화면에서는 Blogger 기본 헤더를 숨기고 `outputs/site.json` 소개와 `outputs/tools.json` 검색을 콘텐츠 앞에 삽입한다.
- 메인 도구 모음은 검색어가 비어 있어도 기존 3열 카드 전체를 표시하고 검색 시 같은 카드 구조로 필터링한다.
- 내부 화면 검색은 검색어가 비어 있으면 링크를 숨기고 입력 뒤에만 키워드·해시태그·유사 검색 결과를 표시한다.
- 게시글의 좌측 하단 도구 검색 복귀 버튼은 기존대로 유지한다.
- 기존 검색 결과 목록의 DOM과 링크 동작은 바꾸지 않고 공통 검색 영역만 위에 별도로 삽입한다.
- 전역 푸터는 기존 위젯과 링크를 유지하면서 낮은 높이, 얇은 구분선, 정돈된 작은 글자와 링크 간격으로 통일한다.

## 검증

- 메인, 게시글, 라벨 검색, 검색 결과 화면을 모바일과 PC에서 각각 확인한다.
- 내부 화면 모두 같은 소개·검색 UI를 사용하며 기본 Blogger 헤더가 중복되지 않는지 확인한다.
- 빈 검색 결과 숨김, 키워드 검색, 가로 넘침을 확인한다.
- 기존 검색 결과 목록이 손상되지 않았는지와 푸터의 여백·정렬·링크 가독성을 확인한다.
