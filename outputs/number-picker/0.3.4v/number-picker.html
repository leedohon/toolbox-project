숫자뽑기

원하는 숫자 범위와 개수를 정하고 중복 여부까지 선택해 무작위 숫자를 뽑아 보세요.

  • 정수 범위와 로또·주사위 빠른 설정
  • 결과 코드 공유와 같은 숫자 복원
  • 최대 100개 결과와 복사
  • KOR / ENG 화면 전환과 PNG 저장

숫자 뽑기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 범위와 개수를 입력하거나 빠른 설정을 선택합니다.
  2. 숫자를 뽑고 결과를 확인합니다.
  3. 결과 코드를 복사해 보내거나 받은 코드를 붙여넣고 같은 숫자를 엽니다.

숫자뽑기 상세 설명

숫자뽑기는 최소값과 최대값 사이에서 원하는 개수만큼 정수를 무작위로 선택합니다. 추첨 순서, 자리 번호, 샘플 번호처럼 범위가 정해진 숫자를 빠르게 고를 때 사용할 수 있습니다.

중복 허용 여부를 선택할 수 있고 결과 코드를 이용해 같은 추첨 결과를 다시 열 수 있습니다. 결과는 복사하거나 PNG로 저장할 수 있습니다.

자주 묻는 질문

로또 번호처럼 중복 없이 뽑을 수 있나요?

네. 중복 허용을 끄고 1부터 45까지, 6개처럼 범위와 개수를 설정하면 됩니다.

음수도 사용할 수 있나요?

네. 허용 범위 안에서 최소값과 최대값에 음수를 입력할 수 있습니다.

뽑을 개수가 숫자 범위보다 크면 어떻게 되나요?

중복을 허용하지 않은 경우에는 뽑을 개수를 범위 안의 전체 숫자 개수 이하로 줄여야 합니다. 입력 가까이에 수정 방법이 표시됩니다.

같은 결과를 다른 사람에게 보낼 수 있나요?

네. 결과 코드를 복사해 보내면 상대방이 코드로 같은 결과를 열 수 있으며 PNG 저장도 지원합니다.
