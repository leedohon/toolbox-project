숫자뽑기

원하는 숫자 범위와 개수를 정하고 중복 여부까지 선택해 무작위 숫자를 뽑아 보세요.

  • 정수 범위와 개수 설정
  • 로또 6/45·주사위 빠른 설정
  • 최대 100개 결과와 복사
  • KOR / ENG 화면 전환

숫자 뽑기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 최솟값과 최댓값을 입력합니다.
  2. 뽑을 개수와 중복·정렬 옵션을 선택합니다.
  3. 숫자를 뽑고 필요하면 결과를 복사합니다.

자주 묻는 질문

로또 번호처럼 중복 없이 뽑을 수 있나요?

네. 중복 허용을 끄고 1부터 45까지, 6개처럼 설정하면 됩니다.

음수도 사용할 수 있나요?

네. -1,000,000부터 1,000,000 사이의 정수를 사용할 수 있습니다.

업데이트 때 게시글을 다시 수정해야 하나요?

아니요. 고정된 iframe 주소에서 최신 기능을 불러옵니다.
