초간단 툴박스

돌림판

메뉴, 벌칙, 순서처럼 고민되는 항목을 넣고 돌림판으로 재미있게 하나를 골라 보세요.

  • 2~20개 항목 입력
  • 자동 색상과 회전 애니메이션
  • 브라우저 난수를 사용한 결과 선택

돌림판 돌리기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 후보 항목을 한 줄에 하나씩 입력합니다.
  2. 항목 반영으로 돌림판을 확인합니다.
  3. 돌림판을 돌리고 선택된 결과를 확인합니다.

자주 묻는 질문

같은 항목을 여러 번 넣을 수 있나요?

네. 같은 항목을 여러 줄에 입력하면 그만큼 선택될 비중이 커집니다.

입력 항목이 서버로 전송되나요?

아니요. 돌림판 생성과 추첨은 현재 브라우저 안에서만 진행됩니다.

업데이트 때 게시글을 다시 수정해야 하나요?

아니요. 고정된 iframe 주소에서 최신 기능을 불러옵니다.
