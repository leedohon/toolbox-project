사용 방법

  1. 인코딩 또는 디코딩 방식을 선택합니다.
  2. HTML 조각이나 엔티티 문자열을 입력합니다.
  3. 변환 결과를 확인하고 복사합니다.

HTML 엔티티 변환기 상세 설명

HTML 엔티티 변환기는 태그로 해석될 수 있는 꺾쇠와 앰퍼샌드, 따옴표를 문자 데이터로 안전하게 표현합니다. 코드 예시를 글에 표시하거나 HTML 속성값을 점검할 때 유용합니다.

디코딩은 브라우저의 텍스트 영역 파서를 이용해 엔티티를 실제 문자로 되돌립니다. 결과를 실행하지 않고 텍스트로만 표시하며 입력은 현재 브라우저 안에서 처리합니다.

자주 묻는 질문

어떤 문자를 인코딩하나요?

앰퍼샌드, 작은·큰 꺾쇠, 큰따옴표와 작은따옴표를 대표 엔티티로 바꿉니다.

HTML 태그가 실행되나요?

아니요. 입력과 결과를 텍스트로만 다룹니다.

숫자 엔티티도 디코딩되나요?

네. 브라우저가 지원하는 이름·10진수·16진수 엔티티를 문자로 되돌립니다.

코드 전체를 붙여넣어도 되나요?

입력 제한 안에서는 가능하며 결과는 브라우저에서만 처리됩니다.
