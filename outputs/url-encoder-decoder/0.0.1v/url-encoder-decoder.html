URL 인코더·디코더

한글, 공백, 특수문자가 포함된 주소나 문장을 URL에 안전한 형식으로 바꾸고 다시 원문으로 되돌리세요.

  • URL 인코딩·디코딩
  • 쿼리 값과 주소 전체 처리 구분
  • 입력·결과 바꾸기와 복사
  • KOR / ENG 화면 전환

URL 변환하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 인코딩 또는 디코딩을 선택합니다.
  2. 쿼리 값·문장이나 주소 전체 중 처리 범위를 고릅니다.
  3. 내용을 입력하고 변환한 뒤 결과를 복사합니다.

자주 묻는 질문

쿼리 값과 주소 전체의 차이는 무엇인가요?

쿼리 값은 ?, &, = 같은 예약문자도 변환합니다. 주소 전체는 주소 구조에 필요한 예약문자를 유지합니다.

입력 내용이 서버로 전송되나요?

아니요. 인코딩과 디코딩은 현재 브라우저 안에서만 처리됩니다.
