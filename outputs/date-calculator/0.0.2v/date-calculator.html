두 날짜 사이가 며칠인지 확인하거나 기준일에서 원하는 일수를 더하고 빼세요.

  • 두 날짜 사이 일수 계산
  • 종료일 포함 여부 선택
  • 날짜 더하기·빼기와 요일 표시
  • 결과 복사와 KOR / ENG

날짜 계산하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 두 날짜 사이 또는 날짜 더하기·빼기를 선택합니다.
  2. 날짜와 필요한 일수를 입력합니다.
  3. 계산 결과와 요일을 확인하고 필요하면 복사합니다.

날짜 계산기 상세 설명

날짜 계산기는 두 날짜 사이의 실제 일수 차이를 확인하거나 기준일에서 원하는 날짜를 빠르게 찾을 때 사용합니다. 마감일, 여행 일정, 보증 기간, 프로젝트 기간을 계산할 때 달마다 다른 일수를 직접 세지 않아도 됩니다.

종료일 포함 여부를 선택할 수 있으며 날짜 더하기·빼기에는 양수와 음수를 모두 사용할 수 있습니다. 계산은 시간대의 영향을 줄이도록 날짜 단위로 처리합니다.

자주 묻는 질문

시작일과 종료일이 같으면 며칠인가요?

기본 결과는 0일입니다. 종료일 포함을 선택하면 시작일 하루를 포함해 1일로 계산합니다.

날짜를 빼려면 어떻게 입력하나요?

날짜 더하기·빼기 모드에서 -30처럼 음수를 입력하면 기준일보다 앞선 날짜를 계산합니다.

윤년의 2월 29일도 반영되나요?

네. 브라우저의 표준 날짜 계산을 사용해 윤년과 월별 일수 차이를 자동으로 반영합니다.

휴대폰에서도 같은 결과를 얻을 수 있나요?

네. 모바일과 PC에서 같은 날짜 계산 규칙을 사용하며 별도 설치 없이 브라우저에서 실행됩니다.
