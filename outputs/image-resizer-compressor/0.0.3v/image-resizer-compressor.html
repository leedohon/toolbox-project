이미지 크기·용량 변환기 상세 설명

이미지 크기·용량 변환기는 블로그, 메신저, 문서에 올릴 사진이 너무 크거나 무거울 때 사용합니다. 가로와 세로 중 하나를 바꾸면 원본 비율에 맞춰 다른 값이 계산되어 찌그러짐을 줄입니다.

JPEG와 WebP는 품질 슬라이더로 용량과 선명도의 균형을 조절할 수 있고 PNG는 투명 배경을 유지합니다. 모든 처리와 미리보기는 현재 브라우저 안에서만 진행됩니다.

자주 묻는 질문

이미지가 서버로 업로드되나요?

아니요. 파일 읽기, 크기 변경과 저장 파일 생성은 현재 브라우저 안에서만 처리합니다.

어떤 형식을 지원하나요?

PNG, JPEG, WebP 입력과 출력을 지원합니다.

PNG에는 품질 설정이 왜 없나요?

PNG는 무손실 형식이라 JPEG나 WebP와 같은 품질 비율을 적용하지 않습니다.

용량이 오히려 커질 수도 있나요?

네. 픽셀 크기를 키우거나 무손실 PNG로 바꾸면 결과 용량이 늘어날 수 있습니다.
