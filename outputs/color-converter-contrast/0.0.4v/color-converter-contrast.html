색상 변환과 대비 확인하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 글자색과 배경색을 색상 선택기 또는 HEX 값으로 입력합니다.
  2. 미리보기와 대비 비율, AA·AAA 판정을 확인합니다.
  3. 필요하면 두 색상을 바꾸거나 변환값을 복사합니다.

색상 변환·대비 검사기 상세 설명

색상 변환·대비 검사기는 웹페이지, 카드뉴스, 문서에서 글자가 배경 위에 충분히 또렷하게 보이는지 확인합니다. 두 색상을 선택하면 대비 비율과 일반 글자·큰 글자의 WCAG 기준을 즉시 표시합니다.

같은 색상을 HEX, RGB와 HSL 형식으로 함께 확인할 수 있어 디자인 도구와 CSS 사이에서 값을 옮길 때 편리합니다. 대비 기준 통과 여부는 보조 지표이며 실제 글꼴 크기와 굵기도 함께 확인해야 합니다.

자주 묻는 질문

AA와 AAA는 무엇인가요?

웹 콘텐츠 접근성에서 사용하는 대비 수준입니다. AAA가 더 높은 대비를 요구합니다.

일반 글자와 큰 글자의 기준이 다른가요?

네. 큰 글자는 획이 더 잘 보여 일반 글자보다 낮은 대비 비율도 허용됩니다.

HEX 값을 직접 입력할 수 있나요?

네. # 뒤에 여섯 자리 16진수 색상을 입력하면 선택기와 결과가 함께 바뀝니다.

투명색도 검사할 수 있나요?

현재 버전은 불투명한 두 색상의 대비만 계산합니다.
