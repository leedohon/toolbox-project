URL 분석·쿼리 편집기

URL의 주소 구성요소와 쿼리 항목을 나누어 확인하고 수정한 주소를 다시 만듭니다.

  • 프로토콜·호스트·경로 분석
  • 쿼리 항목별 편집
  • 중복 쿼리 키 보존
  • 완성 URL 복사

URL 분석하고 다시 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. http 또는 https URL을 입력합니다.
  2. 분석된 구성요소와 번호가 있는 쿼리 행을 확인합니다.
  3. 값을 수정하거나 행을 추가한 뒤 완성 URL을 복사합니다.

URL 분석·쿼리 편집기 상세 설명

URL 분석·쿼리 편집기는 긴 주소를 프로토콜, 호스트, 포트, 경로, 해시와 쿼리 항목으로 나누어 보여 줍니다. 같은 키가 여러 번 쓰인 쿼리도 입력 순서대로 개별 행에 보존합니다.

쿼리 이름과 값은 URLSearchParams 규칙으로 안전하게 인코딩해 다시 조립합니다. 로그인 정보가 포함된 주소는 받지 않으며 모든 분석과 편집은 현재 브라우저에서만 실행합니다.

자주 묻는 질문

상대 주소도 분석할 수 있나요?

현재는 오해를 줄이기 위해 http 또는 https로 시작하는 전체 URL만 지원합니다.

같은 쿼리 이름이 여러 번 있어도 되나요?

네. 각 항목을 별도 행으로 유지해 순서와 중복 키를 보존합니다.

한글 쿼리 값은 어떻게 처리되나요?

편집 결과를 만들 때 브라우저 표준 방식으로 안전하게 퍼센트 인코딩합니다.

입력한 URL이 서버로 전송되나요?

아니요. 주소 분석과 재구성은 현재 브라우저 안에서만 처리합니다.
