평균·중앙값 계산기 상세 설명

평균·중앙값 계산기는 여러 숫자의 대표값과 분포 범위를 빠르게 요약합니다. 평균은 전체 합계를 개수로 나누고 중앙값은 크기순으로 정렬한 가운데 값을 사용합니다.

자주 묻는 질문

평균과 중앙값은 어떻게 다른가요?

평균은 합계를 개수로 나눈 값이고 중앙값은 정렬했을 때 가운데 위치한 값입니다.

숫자를 어떻게 구분하나요?

쉼표, 공백 또는 줄바꿈을 사용할 수 있습니다.

같은 숫자가 여러 번 있어도 되나요?

네. 입력된 횟수만큼 모두 계산에 포함합니다.

소수와 음수도 계산되나요?

네. 일반적인 소수와 음수를 지원합니다.
