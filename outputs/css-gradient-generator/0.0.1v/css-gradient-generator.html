  • 2~5개 색상 지점 추가·삭제
  • 각도와 색상 위치 조절
  • 실시간 미리보기와 CSS 복사

그라데이션 미리보기와 CSS 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 선형 또는 원형 그라데이션 유형을 선택합니다.
  2. 색상 지점을 추가하고 색상·위치·각도를 조절합니다.
  3. 미리보기를 확인한 뒤 CSS background 코드를 복사합니다.

CSS 그라데이션 생성기 상세 설명

CSS 그라데이션 생성기는 웹 배경, 버튼, 카드에 사용할 색상 흐름을 화면에서 조절하고 바로 CSS 코드로 옮길 때 사용합니다. 선형과 원형 방식을 선택하고 각 색상 지점의 위치를 직접 정할 수 있습니다.

색상 지점은 번호가 있는 개별 행과 추가·삭제 버튼으로 관리해 줄바꿈 입력보다 실수를 줄였습니다. 모든 계산과 미리보기는 현재 브라우저에서 진행되며 생성된 background 값은 최신 브라우저의 표준 CSS 문법을 사용합니다.

자주 묻는 질문

색상은 몇 개까지 넣을 수 있나요?

최소 2개부터 최대 5개까지 색상 지점을 추가할 수 있습니다.

색상 위치는 어떻게 정하나요?

각 색상 행의 위치 슬라이더를 0~100% 사이로 움직이면 미리보기와 CSS가 즉시 바뀝니다.

선형과 원형의 차이는 무엇인가요?

선형은 지정한 각도 방향으로 색상이 흐르고 원형은 중심에서 바깥쪽으로 퍼집니다.

만든 CSS는 어디에 넣나요?

복사한 background 선언을 CSS 선택자의 중괄호 안에 붙여 넣으면 됩니다.
