사다리게임

참가자와 결과를 입력해 실제 사다리 경로로 무작위 매칭을 만들어 보세요.

  • 2~12명 사다리 생성
  • 실제 연결선과 경로 계산
  • 결과 코드 공유와 같은 사다리 복원
  • KOR / ENG 화면 전환과 PNG 저장

사다리게임 하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 각 줄에 참가자와 결과를 한 쌍으로 입력합니다.
  2. 사다리를 만들고 연결된 결과를 확인합니다.
  3. 결과 코드를 복사해 보내거나 받은 코드를 붙여넣고 같은 결과를 엽니다.

사다리게임 상세 설명

사다리게임은 참가자와 결과를 같은 수로 입력하고 무작위 연결선을 만들어 한 명씩 결과에 대응시키는 놀이 도구입니다. 순서 정하기, 역할 배정, 간단한 벌칙 정하기에 사용할 수 있습니다.

결과 코드를 복사하면 같은 구성을 다른 기기에서 다시 열 수 있고, 결과 화면은 PNG로 저장할 수 있습니다. 이름과 결과는 서버에 저장되지 않습니다.

자주 묻는 질문

참가자와 결과의 개수는 같아야 하나요?

네. 각 참가자가 하나의 결과를 받도록 두 목록의 행 수를 같게 맞춰 주세요.

같은 참가자가 계속 같은 결과를 받나요?

아니요. 새로 만들거나 다시 섞을 때마다 연결선과 결과 배정이 달라질 수 있습니다.

결과 코드는 무엇인가요?

현재 참가자와 결과 구성을 다시 열 수 있는 짧은 공유용 코드입니다. 코드를 받은 사람은 입력창에 붙여 넣어 같은 결과를 확인할 수 있습니다.

입력한 이름이 서버에 저장되나요?

아니요. 입력과 추첨은 현재 브라우저에서만 처리됩니다.
