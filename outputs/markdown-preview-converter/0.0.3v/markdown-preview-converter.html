  • 안전한 HTML 변환

마크다운 미리보기와 HTML 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 마크다운 문법으로 문서를 입력하거나 붙여넣습니다.
  2. 실시간 미리보기에서 제목, 목록, 링크와 코드 표시를 확인합니다.
  3. 필요하면 변환된 HTML을 복사하거나 파일로 저장합니다.

마크다운 미리보기·HTML 변환기 상세 설명

마크다운 미리보기·HTML 변환기는 README, 블로그 초안, 메모처럼 마크다운으로 작성한 글이 실제로 어떻게 보이는지 빠르게 확인할 때 사용합니다. 제목, 굵게, 기울임, 링크, 인용, 순서 목록과 코드 블록을 실시간으로 변환합니다.

입력한 원문과 변환은 현재 브라우저 안에서만 처리됩니다. 미리보기에는 변환기가 직접 만든 안전한 태그만 사용하고 스크립트나 이벤트 속성은 실행하지 않아, 외부에서 가져온 문서를 검토할 때도 불필요한 코드 실행을 줄입니다.

#초간단툴박스#무료온라인도구#웹도구#마크다운#MarkdownPreview#HTML변환#문서미리보기

자주 묻는 질문

어떤 마크다운 문법을 지원하나요?

제목, 굵게, 기울임, 링크, 인라인 코드, 코드 블록, 인용문과 순서·비순서 목록을 지원합니다.

입력한 글이 서버에 저장되나요?

아니요. 입력, 미리보기와 HTML 변환은 현재 브라우저 안에서만 진행됩니다.

HTML 태그를 직접 입력할 수 있나요?

직접 입력한 HTML은 코드로 처리하며 실행하지 않습니다. 변환기가 생성한 안전한 태그만 미리보기에 사용합니다.

변환 결과를 파일로 받을 수 있나요?

네. 변환된 본문 HTML을 .html 파일로 저장할 수 있습니다.
