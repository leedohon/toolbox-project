초간단 툴박스

돌림판

메뉴, 벌칙, 순서처럼 고민되는 항목을 넣고 돌림판으로 재미있게 하나를 골라 보세요.

  • 2~20개 항목과 회전 추첨
  • 결과 코드 공유와 같은 당첨 결과 복원
  • 브라우저 안에서 안전한 결과 처리
  • KOR / ENG 화면 전환과 PNG 저장

돌림판 돌리기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 각 입력칸에 후보 항목을 하나씩 입력합니다.
  2. 돌림판을 돌리고 선택된 결과를 확인합니다.
  3. 결과 코드를 복사해 보내거나 받은 코드를 붙여넣고 같은 결과를 엽니다.

돌림판 상세 설명

돌림판은 후보 항목을 개별 행으로 추가하고 화면의 원판을 돌려 하나를 무작위로 선택하는 놀이 도구입니다. 메뉴 고르기, 발표 순서, 간단한 경품 추첨처럼 선택 과정을 눈으로 보여 주고 싶을 때 유용합니다.

같은 항목을 여러 번 넣어 선택 비중을 높일 수 있으며 결과 코드를 공유해 같은 구성을 다시 열 수 있습니다. 최종 결과는 PNG로 저장할 수 있습니다.

자주 묻는 질문

같은 항목을 여러 번 넣을 수 있나요?

네. 같은 항목을 여러 행에 입력하면 입력된 횟수만큼 선택될 비중이 커집니다.

빈 항목이 있으면 어떻게 되나요?

빈 행을 표시하고 내용을 입력하도록 안내합니다. 사용하지 않을 행은 삭제 버튼으로 제거해 주세요.

돌린 결과를 다른 사람과 공유할 수 있나요?

네. 결과 코드를 복사해 보내면 같은 항목과 결과를 다시 열 수 있고 PNG 저장도 지원합니다.

입력한 항목이 서버로 전송되나요?

아니요. 돌림판 구성과 추첨은 현재 브라우저 안에서 처리됩니다.
