비율 간단히 계산기 상세 설명

비율 간단히 계산기는 두 수를 최대공약수로 나눠 같은 관계를 유지하는 가장 작은 정수 비로 바꿉니다. 화면 크기, 재료 배합, 수량 비교를 간단히 적을 때 유용합니다.

한쪽 값이 0인 비율도 처리하지만 두 값이 모두 0이면 관계를 정할 수 없어 오류로 안내합니다. 계산은 브라우저에서만 수행됩니다.

자주 묻는 질문

1920:1080은 어떻게 되나요?

최대공약수 120으로 나눠 16:9가 됩니다.

소수 비율도 되나요?

현재는 정확한 최대공약수 계산을 위해 정수 입력에 집중합니다.

한쪽이 0이어도 되나요?

네. 다만 두 값이 모두 0이면 비율을 만들 수 없습니다.

음수도 입력할 수 있나요?

계산은 가능하지만 실제 길이와 수량 비율에는 보통 0 이상의 값을 사용합니다.
