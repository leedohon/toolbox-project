이미지 모자이크 도구

사진에서 얼굴, 이름, 번호처럼 숨기고 싶은 부분을 직접 드래그해 모자이크 처리하세요.

  • 파일 선택과 드래그 앤 드롭
  • 브러시 크기·모자이크 강도 조절
  • 되돌리기·초기화·PNG 저장
  • KOR / ENG 화면 전환

이미지 모자이크 하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. PNG, JPEG 또는 WebP 이미지를 선택하거나 드롭합니다.
  2. 브러시 크기와 강도를 조절하고 숨길 부분을 드래그합니다.
  3. 결과를 확인한 뒤 PNG 파일로 저장합니다.

자주 묻는 질문

이미지가 서버로 업로드되나요?

아니요. 이미지 불러오기, 모자이크 처리, 저장은 모두 현재 브라우저 안에서만 진행됩니다.

모자이크를 잘못 그렸다면 어떻게 하나요?

한 단계 되돌리기를 누르거나 원본으로 초기화한 뒤 다시 작업할 수 있습니다.

휴대폰에서도 사용할 수 있나요?

네. 휴대폰에서 사진을 선택하고 손가락으로 드래그해 모자이크를 적용할 수 있습니다.

업데이트 때 게시글을 다시 수정해야 하나요?

아니요. 고정된 iframe 주소에서 최신 기능을 불러옵니다.
