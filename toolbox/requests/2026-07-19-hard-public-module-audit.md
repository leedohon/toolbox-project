# 2026-07-19 공개 모듈 전수조사 하드 워크플로

## 확정 요청

- 작업 종류: 하드 자동 작업 처리
- 대상: `outputs/tools.json`에 공개된 전체 도구
- 1차 점검: UI 깨짐, 가로 넘침, 잘림, 중복 스크롤, 버튼 줄바꿈, 빈 상태 노출, 입력·오류·결과 동작, KOR / ENG, 런타임 오류
- 입력 UX: 비교 원문과 수정문처럼 긴 텍스트 영역이 둘 이상이면 좌우 분할을 사용하지 않고 위/아래 한 열로 고정
- 지속 규칙: 이후 신규·수정 도구에도 같은 장문 입력 배치 원칙과 회귀 검사를 적용
- Blogger 게시: 완료된 변경만 공개 반영
- Git·배포: `main` 커밋·푸시와 GitHub Pages 공개 확인
- 종료 예약: 사용하지 않음

## 하드 작업량

- 공개 도구 개선 6건
- 신규 또는 통합 페이지 작업 3건
- 공통 결함 수정 2건
- 최근 작업 도구에 편중되지 않도록 공개 도구를 순환하고 은퇴한 주소의 호환 진입점을 보존

## 전수조사로 확정한 작업

### 공개 도구 개선 6건

1. `text-diff-checker`: 두 장문 입력을 데스크톱에서도 위/아래 전체 폭으로 고정한다.
2. `markdown-preview-converter`: 장문 편집칸과 미리보기를 위/아래 전체 폭으로 바꾼다.
3. `qr-barcode-generator`: 비어 있는 문서 제목을 채워 탭과 보조기술에서 도구를 식별하게 한다.
4. `json-tools`: 비어 있는 문서 제목을 채우고 직접 입력에도 5MB 상한을 적용한다.
5. `base64-tools`: 복사 차단 시 모바일에서 결과를 강제로 선택하는 동작을 없애고 직접 선택 가능한 대체 영역만 표시한다.
6. `unicode-inspector`: 넓은 결과 표를 키보드로도 가로 스크롤할 수 있게 이름과 포커스 대상을 제공한다.

### 신규·통합 페이지 작업 3건

1. XML 문법 검사·정리·압축 도구를 신규 공개한다.
2. URL 구성요소와 쿼리 항목을 분석·재구성하는 도구를 신규 공개한다.
3. 중복된 데이터 단위 변환 기능을 `date-calculator`에 모든 SI·IEC 결과 표 형태로 통합하고 `data-size-converter` 기존 주소는 호환 안내로 보존한다.

### 공통 결함 수정 2건

1. 공통 장문 입력 한 열 클래스와 `PORTRAIT_LONG_TEXT_SQUEEZE` 회귀 계약을 추가한다.
2. 공개 카탈로그 전체를 같은 구조 기준으로 반복 검사하는 자동 감사 스크립트와 전수조사 보고서를 추가한다.

## 전수조사 판정 기준

1. 공개 카탈로그와 최신 버전 파일이 일치하고 모든 고정 embed 주소가 열린다.
2. 모바일 세로 375px와 데스크톱 1280px에서 가로 넘침·폭 축소·잘림·중복 스크롤이 없다.
3. 숨겨진 빈 결과나 미리보기가 초기 화면에 노출되지 않는다.
4. 입력, 실행, 초기화, 오류 수정, 복사·저장 등 핵심 흐름이 동작한다.
5. 장문 텍스트 입력 영역이 둘 이상인 도구는 위/아래 순서와 전체 폭을 유지한다.
6. 모바일 터치 환경에서 초기화·오류·항목 추가 뒤 자동 포커스 또는 자동 선택이 발생하지 않는다.
7. KOR / ENG 선택이 정적 문구와 동적 상태 모두에 적용된다.
8. 콘솔 오류와 오래된 공통 자산 참조가 없고 iframe 높이가 콘텐츠 증감에 맞춰 갱신된다.

## 기록 방식

- 28개 공개 도구별 통과 항목과 발견 결함을 전수조사 보고서에 남긴다.
- 실제 수정은 버전 보존 절차에 따라 새 릴리스에서만 수행한다.
- 수정한 결함은 재발 방지 레지스트리와 자동 검사에 연결한다.
