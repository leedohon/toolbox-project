URL 슬러그 생성기 상세 설명

URL 슬러그는 게시글이나 상품 페이지 주소의 마지막 경로를 짧고 읽기 좋게 표현합니다. 이 도구는 대문자를 소문자로 바꾸고 연속된 공백과 기호를 하나의 구분자로 정리합니다.

한글과 영문, 숫자는 유지하며 앞뒤 구분자는 제거합니다. 결과는 현재 브라우저에서만 생성되므로 입력한 제목이나 내부 문구가 외부로 전송되지 않습니다.

자주 묻는 질문

한글도 슬러그에 남나요?

네. 한글·영문·숫자를 유지하고 공백과 기호만 선택한 구분자로 정리합니다.

영문 대문자는 어떻게 되나요?

주소를 일관되게 만들기 위해 소문자로 바뀝니다.

공백이 여러 개면 어떻게 되나요?

연속된 공백과 기호를 하나의 하이픈 또는 밑줄로 합칩니다.

입력한 제목이 서버로 전송되나요?

아니요. 모든 정리는 현재 브라우저에서 처리합니다.
