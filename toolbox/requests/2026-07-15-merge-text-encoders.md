# Base64 · URL 변환 도구 통합

## 요청

- Base64 인코더·디코더와 URL 인코더·디코더를 하나의 기능으로 통합한다.
- 의미가 겹치는 별도 도구 카드는 공개 목록에서 제외한다.
- 기존 URL 도구의 모바일 화면 잘림을 통합 화면에서 해결한다.
- 모바일 환경에서는 변환 형식 변경, 초기화, 복사 실패 뒤 입력칸에 자동 포커스하지 않는다.

## 구현 결정

- `base64-tools`를 Base64와 URL 형식을 선택하는 텍스트 인코더·디코더로 확장한다.
- URL은 쿼리 값·문장과 주소 전체 처리 범위를 유지한다.
- `url-encoder-decoder`는 `status: retired`로 카탈로그에서 제외한다.
- 기존 URL 게시글과 고정 주소는 삭제하지 않고 통합 도구를 안내하는 호환 진입점으로 유지한다.
- 통합 실행 화면은 공통 스카이블루 테마와 세로 입력·결과 구조를 사용하며 360px에서 가로 넘침이 없어야 한다.

## 검증

- Base64 UTF-8 왕복 변환과 URL 구성요소·전체 주소 왕복 변환을 확인한다.
- 360px 모바일과 데스크톱에서 가로 잘림, 버튼 줄바꿈, 입력 영역 폭을 확인한다.
- 모바일 초기화 뒤 입력칸이 자동 활성화되지 않는지 확인한다.
- 공개 카탈로그에서 중복 URL 도구 카드가 제거되고 기존 URL 게시글이 통합 도구로 연결되는지 확인한다.
