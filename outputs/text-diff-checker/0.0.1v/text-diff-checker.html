  • 변경 수와 일치율 요약

사용 방법

  1. 왼쪽에 원본, 오른쪽에 수정한 텍스트를 입력합니다.
  2. 필요하면 공백 또는 영문 대소문자 무시를 선택합니다.
  3. 색상으로 구분된 차이와 요약을 확인하거나 결과를 복사합니다.

텍스트 비교·차이 확인기 상세 설명

텍스트 비교·차이 확인기는 문서, 공지, 코드 조각처럼 비슷한 두 텍스트에서 바뀐 줄을 찾을 때 사용합니다. 원본에서 사라진 줄과 수정본에 추가된 줄을 구분하고, 그대로 유지된 줄도 함께 보여 전체 흐름을 놓치지 않게 합니다.

비교는 현재 브라우저 안에서만 처리되며 입력 내용은 전송되지 않습니다. 공백과 영문 대소문자 차이를 무시하는 옵션을 사용하면 내용상 중요한 변경에 집중할 수 있고, 긴 문서는 입력 길이 제한 안에서 처리합니다.

자주 묻는 질문

어떤 기준으로 차이를 찾나요?

줄바꿈을 기준으로 원본과 수정본의 공통 줄을 찾고 추가·삭제된 줄을 표시합니다.

띄어쓰기 차이를 무시할 수 있나요?

네. 공백 무시를 선택하면 연속 공백과 줄 앞뒤 공백 차이를 제외하고 비교합니다.

입력한 문서가 서버로 전송되나요?

아니요. 비교와 결과 생성은 현재 브라우저 안에서만 진행됩니다.

아주 긴 문서도 비교할 수 있나요?

브라우저가 느려지는 것을 막기 위해 각 입력은 5만 자까지 지원합니다.
