CSS 디자인 도구 상세 설명

CSS 디자인 도구는 색상 변환·대비 검사, 선형·원형 그라데이션, 반응형 clamp 값, box-shadow, px·rem·em 단위와 border-radius 코드를 한 화면에서 선택해 만드는 통합 도구입니다. 웹 화면을 꾸밀 때 자주 쓰는 값을 여러 생성기로 나누어 찾지 않고 필요한 모드만 바꿔 사용할 수 있습니다.

색상 모드는 HEX·RGB·HSL 변환과 WCAG 대비를 확인하고, 그라데이션·그림자·모서리 모드는 미리보기를 보며 CSS를 생성합니다. clamp 모드는 최소·최대 크기와 뷰포트 범위로 반응형 식을 계산하고, 단위 변환은 기준 글자 크기에 맞춰 px·rem·em을 바꿉니다.

입력과 미리보기는 현재 브라우저 안에서만 처리되며 서버에 저장되지 않습니다. 색상 형식이나 숫자 범위가 올바르지 않으면 해당 모드의 안내에 따라 값을 수정하고, 복사가 차단되면 화면에 표시되는 대체 결과를 직접 선택해 복사할 수 있습니다.

자주 묻는 질문

어떤 CSS 기능을 지원하나요?

색상 변환·대비, 그라데이션, clamp, box-shadow, px·rem·em 변환과 border-radius 생성을 지원합니다.

색상 대비의 AA와 AAA는 무엇인가요?

웹 콘텐츠 접근성에서 사용하는 대비 수준이며 AAA가 더 높은 대비를 요구합니다. 실제 글꼴 크기와 굵기도 함께 확인하세요.

그라데이션·그림자·모서리 결과를 미리 볼 수 있나요?

네. 색상과 방향, 위치·흐림·확산, 네 모서리 값을 바꾸면 미리보기와 복사할 CSS 코드가 함께 갱신됩니다.

clamp와 px·rem·em 변환에는 어떤 기준이 필요한가요?

clamp는 최소·최대 크기와 뷰포트 범위가 필요하고, 단위 변환은 기준 글자 크기를 사용합니다.

잘못된 값이나 복사 오류는 어떻게 처리하나요?

색상 형식과 숫자 범위를 수정한 뒤 다시 실행하세요. 자동 복사가 차단되면 대체 입력칸에 표시된 CSS를 직접 복사할 수 있습니다.

입력한 디자인 값이 서버에 저장되나요?

아니요. 모든 계산, 변환과 미리보기는 현재 브라우저 안에서만 처리됩니다.
