이미지 크기·압축·DPI 도구는 사진의 픽셀 크기와 저장 용량을 줄이는 변환 모드, 픽셀 수와 실제 출력 크기로 DPI·PPI를 계산하는 해상도 모드를 한곳에 모았습니다. 웹 게시용 최적화와 인쇄용 해상도 확인을 같은 화면에서 이어서 처리할 수 있습니다.

이미지 변환은 원본 비율을 유지한 채 PNG·JPEG·WebP 형식과 품질을 조절하고 결과를 미리 본 뒤 저장합니다. 해상도 계산은 가로·세로 픽셀과 inch·cm 출력 크기를 사용해 각 방향의 DPI·PPI를 보여 주므로 인쇄 크기와 선명도를 비교할 때 유용합니다.

파일 읽기, 미리보기, 변환과 계산은 모두 현재 브라우저 안에서 실행되며 이미지가 서버로 업로드되지 않습니다. 지원하지 않는 파일이거나 크기·출력 단위 값이 올바르지 않으면 안내를 확인해 파일을 다시 선택하거나 숫자를 수정한 뒤 실행하세요.

자주 묻는 질문

이미지 변환과 DPI·PPI 계산은 어떻게 선택하나요?

화면 위쪽에서 크기·용량 변환 또는 해상도 계산 모드를 선택하면 해당 기능에 필요한 입력만 표시됩니다.

어떤 이미지 형식을 지원하나요?

PNG, JPEG와 WebP 입력·출력을 지원하며 JPEG와 WebP는 품질을 조절할 수 있습니다.

DPI와 PPI 계산에는 어떤 값이 필요한가요?

이미지의 가로·세로 픽셀과 실제 출력할 가로·세로 크기, inch 또는 cm 단위가 필요합니다.

결과 용량이 커지거나 계산 오류가 나오면 어떻게 하나요?

이미지는 픽셀 크기나 품질을 낮추고 JPEG·WebP를 선택해 보세요. 해상도 계산은 0보다 큰 픽셀·출력 크기를 입력한 뒤 다시 계산하세요.

이미지가 서버로 업로드되나요?

아니요. 파일 읽기, 크기 변경, 저장 파일 생성과 DPI·PPI 계산은 현재 브라우저 안에서만 처리합니다.
