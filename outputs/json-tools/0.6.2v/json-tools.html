JSON 도구

JSON을 검사하고 정리하거나 표로 미리 본 뒤 Excel·CSV 파일로 저장할 수 있습니다.

  • JSON 유효성과 오류 위치 확인
  • 예제 JSON 불러오기와 들여쓰기 정리
  • 표 미리보기와 JSON·Excel·CSV 저장
  • KOR / ENG 화면 전환

JSON 정리하기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. JSON을 입력창에 붙여 넣습니다.
  2. 정리 결과와 표를 확인합니다.
  3. 결과를 JSON·Excel·CSV로 저장합니다.

JSON 도구 상세 설명

JSON 도구는 API 응답이나 설정 파일의 구조를 검사하고 읽기 쉽게 정리하는 브라우저 도구입니다. 문법 오류가 있으면 줄과 열 위치를 안내해 쉼표, 따옴표, 괄호 문제를 빠르게 찾을 수 있습니다.

유효한 JSON은 보기 좋은 들여쓰기 형식이나 한 줄 형식으로 바꿀 수 있습니다. 배열과 객체는 표로 미리 보고 Excel 또는 CSV 파일로 저장할 수 있습니다.

자주 묻는 질문

입력한 JSON이 서버에 저장되나요?

아니요. 검사, 정리, 표 변환과 파일 생성은 현재 브라우저 안에서 처리됩니다.

오류 위치는 어떻게 확인하나요?

결과 영역의 강조 위치와 오류 문장에 표시된 줄·열 번호를 확인한 뒤 주변의 쉼표, 따옴표와 괄호를 점검하세요.

정리와 압축은 무엇이 다른가요?

정리는 줄바꿈과 들여쓰기를 추가해 읽기 쉽게 만들고, 압축은 불필요한 공백을 제거해 한 줄로 만듭니다.

어떤 JSON을 Excel 표로 만들기 좋나요?

같은 키를 가진 객체 배열이 가장 자연스러운 행·열 표가 됩니다. 중첩 객체는 점으로 연결한 열 이름으로 펼쳐 표시합니다.
