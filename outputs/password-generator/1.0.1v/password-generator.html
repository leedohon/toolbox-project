비밀번호·UUID 생성기 상세 설명

비밀번호·UUID 생성기는 안전한 무작위 비밀번호와 UUID v4 식별자를 한 화면에서 선택해 만드는 도구입니다. 비밀번호 모드에서는 길이와 문자 구성을 정하고, UUID 모드에서는 생성 개수와 하이픈·중괄호 형식을 선택해 여러 값을 한 번에 복사하거나 파일로 저장할 수 있습니다.

두 모드 모두 브라우저의 암호학적 난수 기능을 사용합니다. 생성된 비밀번호는 계정마다 다르게 사용하고 신뢰할 수 있는 비밀번호 관리 앱에 보관하는 것이 좋으며, UUID는 항목을 구분하는 식별자이지 비밀번호나 인증 수단이 아닙니다.

생성 조건과 결과는 현재 브라우저 안에서만 처리되며 서버로 전송하거나 저장하지 않습니다. 선택한 문자군이 없거나 개수·길이가 범위를 벗어나면 안내에 따라 값을 고친 뒤 다시 생성하거나 초기화할 수 있습니다.

자주 묻는 질문

비밀번호와 UUID는 언제 각각 사용하나요?

비밀번호는 계정 인증에 사용하고, UUID는 문서·데이터·작업 항목을 겹치지 않게 구분하는 식별자로 사용합니다.

안전한 비밀번호 길이는 어느 정도인가요?

일반 계정은 16자 이상을 권장하며 서비스가 허용한다면 여러 문자군을 사용한 20자 이상이 더 안전합니다.

UUID를 여러 개 만들고 형식을 바꿀 수 있나요?

네. 원하는 개수를 한 번에 생성하고 기본형, 하이픈 제외형, 중괄호 포함형으로 복사하거나 텍스트 파일로 저장할 수 있습니다.

생성 버튼이 동작하지 않거나 오류가 나오면 어떻게 하나요?

비밀번호 문자군을 하나 이상 선택하고 길이·UUID 개수를 허용 범위로 고친 뒤 다시 실행하세요. 초기화하면 권장 기본값으로 돌아갑니다.

생성 결과가 서버에 저장되나요?

아니요. 비밀번호와 UUID 생성은 현재 브라우저 안에서만 진행되며 결과를 전송하거나 저장하지 않습니다.
