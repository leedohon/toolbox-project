# 모자이크 사각형 선택 도구 추가

## 요청

- 이미지 모자이크 도구에 드래그 시작점과 종료점으로 정한 사각형 범위 전체를 모자이크 처리하는 기능을 추가한다.

## 구현 결정

- 편집 도구는 `브러시`와 `사각형 선택` 두 가지 라디오 선택으로 제공한다.
- 사각형 선택 중에는 파란 점선 미리보기를 표시하고, 포인터를 놓을 때 범위 전체에 모자이크를 적용한다.
- 사각형 한 번의 적용은 되돌리기 기록 한 단계로 저장한다.
- 너비나 높이가 너무 작은 선택은 오조작으로 보고 적용하지 않으며 쉬운 안내를 표시한다.
- 사각형 모드에서는 사용하지 않는 브러시 크기 설정을 숨기고 모자이크 강도만 표시한다.
- 마우스, 터치, 펜 입력에 동일한 포인터 이벤트를 사용한다.
- 기능 추가 수준이므로 `minor` 변경인 `0.2.0v`로 릴리스한다.

## 검증

- 샘플 이미지에서 사각형 드래그 전후 픽셀 변화와 선택 범위 밖 픽셀 보존을 확인한다.
- 사각형 적용 후 한 단계 되돌리기와 원본 초기화를 확인한다.
- 브러시 모드가 기존처럼 동작하는지 회귀 검증한다.
- KOR / ENG 문구, 375px 모바일과 1280px 데스크톱 화면의 넘침 및 조작 영역을 확인한다.
