사용 방법

  1. 분석할 텍스트를 입력합니다.
  2. 문자별 코드 포인트와 표기를 확인합니다.
  3. 필요한 결과를 TSV 형식으로 복사합니다.

유니코드 문자 분석기 상세 설명

유니코드 문자 분석기는 눈으로 비슷해 보이는 문자와 보이지 않는 공백을 코드값으로 구분해 보여 줍니다. 다국어 텍스트, 이모지, 특수문자의 실제 구성을 확인할 때 유용합니다.

브라우저의 코드 포인트 순회 방식을 사용해 기본 다국어 영역 밖의 문자도 한 글자로 처리합니다. 결합 문자와 이모지 조합은 여러 코드 포인트로 표시될 수 있으며 원문은 서버로 전송하지 않습니다.

자주 묻는 질문

이모지가 여러 줄로 표시되는 이유는 무엇인가요?

일부 이모지는 여러 코드 포인트를 결합해 한 모양으로 보이기 때문입니다.

공백도 분석할 수 있나요?

네. 일반 공백, 줄바꿈, 탭, 줄바꿈 없는 공백 등 대표 문자를 이름으로 표시합니다.

UTF-16 값은 왜 두 개일 수 있나요?

U+FFFF를 넘는 문자는 UTF-16에서 서로게이트 두 단위로 표현됩니다.

텍스트가 외부로 전송되나요?

아니요. 입력과 분석 결과는 현재 브라우저에서만 처리합니다.
