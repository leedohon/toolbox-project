  • 텍스트와 파일 입력
  • SHA-256·SHA-384·SHA-512
  • 대문자·소문자 출력
  • 결과 복사

사용 방법

  1. 텍스트를 입력하거나 파일을 선택합니다.
  2. 해시 알고리즘과 출력 형식을 선택합니다.
  3. 계산한 해시값을 복사합니다.

텍스트·파일 해시 생성기 상세 설명

해시는 같은 입력에서 항상 같은 길이의 식별값을 만드는 방식입니다. 다운로드 파일의 무결성을 대조하거나 두 텍스트가 같은지 확인할 때 원문 전체를 비교하지 않고도 사용할 수 있습니다.

입력한 텍스트와 파일은 서버로 전송하지 않고 현재 브라우저의 Web Crypto 기능으로 처리합니다. 해시값은 암호화된 원문이 아니므로 다시 복원할 수 없으며 비밀번호 저장 용도로 단독 사용해서는 안 됩니다.

자주 묻는 질문

파일이 서버로 전송되나요?

아니요. 선택한 파일은 현재 브라우저에서만 읽어 해시를 계산합니다.

어떤 알고리즘을 선택하면 되나요?

일반적인 무결성 확인에는 SHA-256이 널리 쓰이며, 상대방이 지정한 알고리즘이 있으면 같은 항목을 선택하세요.

같은 파일인데 해시가 다를 수 있나요?

파일 내용이 한 바이트라도 달라지면 해시가 달라집니다. 파일명은 계산 결과에 영향을 주지 않습니다.

해시로 원문을 복원할 수 있나요?

아니요. 이 도구의 해시는 원문으로 되돌리는 인코딩이 아닙니다.
