# 게시글 전역 SEO 작성 원칙 추가

## 요청

- 앞으로 새로 작성하거나 수정하는 Blogger 게시글은 검색 의도에 맞는 SEO 친화적 구조로 작성한다.
- 적용 범위는 게시글 제목, 리드, 사용 방법, 상세 설명, FAQ, 링크, 해시태그와 검색 키워드다.
- 실행 iframe, 기능 UI와 도구 로직에는 SEO용 문구를 추가하지 않는다.

## 공통화

- 세부 기준은 `toolbox/seo-rules.json`을 단일 원본으로 관리한다.
- 전역 `AGENTS.md`, `toolbox/AGENTS.md`, 게시글 작성 가이드와 `toolbox-post-draft` 스킬은 적용 범위와 실행 순서만 참조한다.
- 게시글을 수정할 때 검색 의도, 제목·리드 일치, 논리적인 제목 구조, 자연스러운 키워드, 사용자 중심 FAQ, 해시태그·Blogger 라벨·검색어 일치를 확인한다.
- 키워드 반복, 숨김 텍스트, 중복 설명, 관련 없는 분량 늘리기와 운영자용 FAQ는 금지한다.

## 토큰 최적화

- 세부 SEO 규칙을 여러 Markdown과 스킬에 복제하지 않는다.
- 게시글 콘텐츠를 변경할 때만 중앙 JSON을 읽고 기능 전용 변경에서는 읽지 않는다.
