QR · 바코드 생성기

텍스트나 URL을 원하는 색상의 QR 코드 또는 Code 128 바코드로 만들어 저장하세요.

  • 기본형·둥근형 QR 코드
  • 전경색·배경색과 크기 설정
  • 바코드 하단 텍스트 표시와 기본값 초기화
  • KOR / ENG 화면 전환과 PNG·SVG 저장

QR · 바코드 만들기

도구가 보이지 않으면 새 창에서 열기를 이용하세요.

사용 방법

  1. 텍스트 또는 URL을 입력합니다.
  2. 유형·스타일·크기·색상을 설정하고 바코드라면 하단 텍스트 표시 여부를 선택합니다.
  3. 결과를 확인하고 이미지로 저장합니다.

QR · 바코드 생성기 상세 설명

QR 코드는 URL과 긴 텍스트를 카메라로 빠르게 전달할 때 적합하고, Code 128 바코드는 영문·숫자 중심의 식별 값을 스캐너로 읽을 때 적합합니다. 용도에 맞는 형식을 선택해 미리보기를 즉시 만들 수 있습니다.

크기와 QR 모듈 모양을 조절할 수 있고 PNG 또는 SVG로 저장할 수 있습니다. 바코드 아래의 원문 텍스트는 필요에 따라 표시하거나 숨길 수 있습니다.

자주 묻는 질문

QR 코드와 바코드 중 무엇을 사용해야 하나요?

웹 주소나 비교적 긴 텍스트는 QR 코드가 편리하고, 상품 번호나 짧은 식별 값은 Code 128 바코드가 적합합니다.

바코드 아래 글자를 숨길 수 있나요?

네. 바코드 설정에서 하단 텍스트 표시를 끄면 막대만 저장할 수 있습니다.

PNG와 SVG는 어떻게 다른가요?

PNG는 바로 공유하기 좋은 이미지이고, SVG는 크기를 크게 바꿔도 선명해 인쇄와 편집에 적합합니다.

입력 내용이 서버에 저장되나요?

아니요. 코드 생성과 파일 저장은 현재 브라우저 안에서 처리됩니다.
