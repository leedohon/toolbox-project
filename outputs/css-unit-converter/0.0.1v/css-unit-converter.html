CSS px·rem·em 변환기 상세 설명

CSS 단위 변환기는 px 값을 기준 글자 크기로 나누어 rem·em으로 바꾸거나 반대로 곱해 px 값을 구합니다. 디자인 명세와 코드의 단위가 다를 때 빠르게 대조할 수 있습니다.

이 계산은 rem과 em이 같은 기준값을 사용한다고 가정합니다. 실제 em은 요소의 부모 글자 크기에 따라 달라질 수 있으므로 적용 위치의 계산 스타일도 확인하세요.

자주 묻는 질문

16px는 몇 rem인가요?

기준 글자 크기가 16px이면 1rem입니다.

em과 rem은 항상 같은가요?

아니요. rem은 루트, em은 현재 요소 주변의 글자 크기를 기준으로 할 수 있습니다.

기준 글자 크기를 바꿀 수 있나요?

네. 프로젝트가 사용하는 기준 px 값을 입력할 수 있습니다.

소수값도 변환되나요?

네. 소수 입력과 최대 6자리로 정리된 결과를 지원합니다.
